# 《云海山庄》完整 UI 设计规范
**版本**: 1.0
**日期**: 2026-04-21
**分辨率基准**: 1280×720 (PC) / 动态适配 (Switch)
**风格**: 东方治愈系像素风 · 云海水墨质感
**引擎**: C++20 / SFML 3.0.2

---

## 文档说明

本文档是《云海山庄》完整 UI 设计的落地规范，**严格贴合**以下现有资产：

- `PixelArtStyle.hpp` — 像素调色板常量（已定义的颜色全部复用）
- `PixelUiConfig.hpp` — 布局常量（屏幕尺寸、组件位置均基于 1280×720）
- `ui_layout.json` — 面板 JSON 配置（坐标/尺寸/颜色直接引用）
- `PixelGameHud.hpp` — HUD 组件架构（以现有组件为基础扩展）
- `gameplay.cfg` — 游戏参数配置（体力/云海/键位等）

所有尺寸以 **像素 (px)** 为单位，基于 1280×720 设计。Switch 适配通过等比缩放实现。

## 1. 整体 UI 风格定义

### 1.1 视觉关键词

```
宣纸质感 · 云雾缭绕 · 淡墨勾勒 · 玉质温润 · 低饱和治愈
```

界面追求"如置身百年山庄的清晨推窗见云海"——不炫技、不压迫、信息分层清晰。像素风格确保复古感与多平台一致，1px 描边 + 8px 角块装饰保持经典游戏手感。

### 1.3 材质规范

- **宣纸质感**: 面板背景使用 `ColorPalette::Cream` (#FBF4E0) 配合 1px `ColorPalette::BrownOutline` 描边，模拟宣纸卷轴感
- **木纹**: 主要用于设置面板、存档面板的标题栏背景，使用 `ColorPalette::DeepCream` (#E8D8A8)
- **云纹**: 通过 `PixelArtStyle::DrawPixelCorner()` 在面板四角绘制 8×8 纯色角块，象征云海山峦
- **淡墨**: 所有文字使用 `ColorPalette::TextBrown` (#3D2517) 而非纯黑，营造水墨晕染感
- **玉石质感**: 灵茶品质图标、好感进度条使用 `ColorPalette::LightBrown` (#D4A84B) 到 `ColorPalette::DeepBrown` (#8B5A2B) 渐变

### 1.4 圆角/边框/阴影规范

- **圆角**: **0px** — 纯像素风格，不使用圆角
- **像素描边**: 主边框 1px，重点选中边框 2px
- **角块装饰**: 8×8px 纯色块，位于面板四角
- **内阴影**: 同一颜色深 2 度，偏移 1px（通过 `ColorPalette::Darken()` 实现）
- **面板阴影**: 无投影（像素风格保持扁平）

### 1.5 整体氛围描述

> "如清晨推开百年山庄的木窗，云海在山峦间缓缓流动。界面的每一块面板都是宣纸上的淡墨山水——信息清晰可辨，却不打断那份宁静。"

## 2. 全局 UI 组件库

### 2.1 按钮（Button）

**用途**: 菜单选项、功能确认、标签切换

**尺寸规范**:
- 标准按钮: 宽 120px × 高 40px（菜单项）
- 小按钮: 宽 80px × 高 32px（面板内操作）
- 图标按钮: 48×48px（工具栏）

**样式状态**:

| 状态 | 背景 | 边框 | 文字颜色 |
|------|------|------|----------|
| 默认 | `ColorPalette::LightCream` (#F7EFD2) | 1px `ColorPalette::BrownOutline` | `ColorPalette::TextBrown` |
| Hover | `ColorPalette::HighlightYellow` (#FFFACD) | 1px `ColorPalette::BrownOutline` | `ColorPalette::TextBrown` |
| 选中/按下 | `ColorPalette::ActiveGreen` (#90EE90) | 1px `ColorPalette::BrownOutline` | `ColorPalette::TextBrown` |
| 禁用 | `ColorPalette::LightGray` (#C8C0A8) | 1px `ColorPalette::DarkGray` | `ColorPalette::DarkGray` |

**SFML 实现参考**: 继承 `PixelUiPanel`，或直接复用现有面板系统。热区最小 44×44px（Switch 手柄适配）。

```cpp
// 按钮样式枚举
enum class ButtonState { Default, Hover, Pressed, Disabled };

// 按钮尺寸常量（建议加入 PixelUiConfig.hpp）
struct ButtonConfig {
    static constexpr float StandardWidth = 120.0f;
    static constexpr float StandardHeight = 40.0f;
    static constexpr float SmallWidth = 80.0f;
    static constexpr float SmallHeight = 32.0f;
    static constexpr float IconSize = 48.0f;
    static constexpr float HotspotMinSize = 44.0f; // Switch 手柄适配
};
```

### 2.4 图标槽（IconSlot）

**用途**: 物品格、技能格、任务图标

**尺寸**:
- 物品格: 48×48px（背包/工具栏）
- 小图标: 32×32px（标签栏图标）
- 徽章: 20×20px（数量/状态）

**样式**:
- 空槽: 白色 `#FEFEF8` 填充 + 1px `#C8C0A8` 边框
- 有物品: 显示物品贴图，物品数量在右下角 12px 粗体
- 选中: 2px `#90EE90` 绿色描边
- Hover: 1px `#FFFACD` 浅黄高亮

**现有代码参考**: `PixelToolbar` 和 `PixelInventoryGrid` 中的格子系统已实现。

### 2.5 滚动面板（ScrollPanel）

**用途**: 长列表（任务、NPC 对话、契约书卷、存档列表）

**样式**:
- 滚动条: 宽 12px，`ColorPalette::LightGray` 轨道 + `ColorPalette::BrownOutline` 滑块
- 滑块: 最小高度 20px，随内容比例自动计算
- 滚动箭头: 上/下各 16×16px 三角形（像素风格）
- 列表项高度: 32px（任务/存档）、48px（NPC 列表/契约条目）

**交互**:
- 鼠标滚轮滚动
- 手柄: RB/LB 翻页，LS 上下移动
- 点击滚动条轨道跳转

### 2.6 标签页（TabBar）

**用途**: 背包分类、契约分卷、NPC 筛选

**样式**:
- 标签栏高度: 40px
- 每个标签: 图标(24×24) + 文字(14px)，共约 72px 宽
- 选中标签: 底部 3px `ColorPalette::ActiveGreen` 下划线
- 未选中: 无下划线，文字颜色 `#6E6454`

**交互**:
- 点击切换
- 手柄: 左/右方向键切换
- 支持拖拽排序（仅菜单设置界面）

### 2.7 对话框/气泡（DialogueBox）

**用途**: NPC 对话、任务提示、系统通知

**样式**:
- 位置: 底部 1/5 区域 (y=520, h=160)
- 背景: `ColorPalette::BackgroundWhite` (#FEFEF8)，透明度 230/255
- 边框: 1px `ColorPalette::BrownOutline`
- NPC 头像: 左侧 48×48px，像素风格
- 名字: 16px 加粗 `ColorPalette::DeepBrown`
- 文字: 14px `ColorPalette::TextBrown`，行高 1.5 倍，每行 38 字
- 继续提示: 右下角 `▼` 闪烁（0.5s 间隔）
- 选项: Hover 浅黄，Enter 选中绿底

**打字机效果**: 每字符 40ms（可配置），Enter 键跳过全部显示。

**现有代码参考**: `PixelDialogueBox` 已实现打字机、头像、选项、状态机（Hidden/Typing/WaitingChoice/WaitingConfirm）。直接复用。

### 2.8 提示框/气泡（Tooltip）

**用途**: 物品悬停提示、NPC 名字标签、交互提示

**样式**:
- 背景: `ColorPalette::Cream` (#FBF4E0)，透明度 250/255
- 边框: 1px `ColorPalette::BrownOutline`
- 内边距: 8px
- 最大宽度: 200px
- 文字: 12px `ColorPalette::TextBrown`
- 显示延迟: 300ms

**内容格式**:
```
[物品名称]（16px 加粗 深棕）
[品质标签]（12px 彩色）
---
[描述文字]（12px 正文）
[价格/效果]（12px 金色）
```

### 2.9 开关、勾选框（Toggle / Checkbox）

**用途**: 设置界面（音乐/音效/全屏）、工坊自动制作开关

**样式**:
- 尺寸: 32×20px
- OFF: 浅灰轨道 `#C8C0A8`，圆角矩形
- ON: 绿色轨道 `#90EE90`，白色滑块
- 过渡动画: 0.15s 滑块移动

### 2.10 分页（Pagination / PageView）

**用途**: 契约书卷（6 卷，每卷多章节）、存档列表（3 档位）、工坊配方列表

**样式**:
- 页码指示: 小圆点或数字 "1/6"
- 当前页: `ColorPalette::ActiveGreen` 填充
- 非当前页: `ColorPalette::LightGray` 填充
- 翻页箭头: 左/右 24×24px 像素箭头按钮
- 每页高度固定，内容区域滚动

### 2.11 状态栏图标（StatusIcon）

**用途**: HUD 右上角天气/灵气图标、右下角快捷操作提示

**样式**:
- 图标: 24×24px 像素图标
- 文字标注: 12px `ColorPalette::TextBrown`
- 组合: [图标 文字]，间距 4px
- 天气图标: ☀ 晴 / ☁ 薄雾 / ☁☁ 浓云 / ✦ 大潮（使用 SFML Text 或自定义贴图）
- 新任务提示: `❗` 图标，0.5s 闪烁

### 2.12 通知横幅（NotificationBanner）

**用途**: 升级/获得成就/节日开始/大潮预警

**样式**:
- 位置: 屏幕顶部居中，y=16
- 尺寸: 宽 600px × 高 36px
- 背景: `ColorPalette::Cream` + 2px `ColorPalette::LightBrown` 顶部装饰条
- 动画: 从上淡入 (0.3s)，停留 3s，从上淡出 (0.3s)
- 可堆叠显示多个通知（垂直排列，间隔 4px）

### 6.2 绘制层级（从后到前）

`PixelUiConfig.hpp` 中 `RenderLayer` 枚举已定义，渲染顺序：

```
1.  BackgroundScene         — 游戏场景（地图/作物/NPC/玩家）
2.  AuraOverlay             — 灵气/云海色调覆盖
3.  DayNightGlow            — 昼夜光晕
4.  Minimap                 — 迷你地图（M 键）
5.  DialogueBox              — 对话框
6.  Inventory               — 背包（I 键）
7.  QuestMenu               — 任务面板（F 键）
8.  TopRightInfo             — 右上角常驻信息
9.  BottomRightStatus        — 右下角体力/金币
10. Toolbar                 — 快捷栏（底部）
11. WorldTip                — 世界交互提示（头顶）
12. Notification            — 通知横幅
13. DebugInfo               — 调试面板（F3）
```

**对话框优先级**: `DialogueBoxState::Typing/WaitingChoice/WaitingConfirm` 时，其他面板自动隐藏。

### 6.5 动画规范

**允许的动画**（仅以下类型，禁止复杂物理）:

| 动画类型 | 参数 | 说明 |
|----------|------|------|
| 淡入淡出 | 0.15s / 0.10s | 面板打开/关闭 |
| 上浮 | 0.3s ease-out, +10px | 物品获得提示 |
| 闪烁 | 0.5s 周期 | 新任务图标、低体力警告 |
| 打字机 | 40ms/字符 | 对话文字逐字显示 |
| 进度填充 | 即时 | 体力/灵气条变化（无过渡） |
| 焦点呼吸 | 0.3s ease-in-out, ±2px | 手柄焦点元素轻微呼吸 |

**动画实现示例**:
```cpp
// 面板淡入淡出（PixelUiPanel 已实现）
void PixelUiPanel::FadeIn(float duration_seconds) {
    anim_state_ = AnimationState::FadeIn;
    anim_duration_ = duration_seconds;
    anim_elapsed_ = 0.0f;
}

// 低体力警告闪烁（PixelProgressBar 已实现）
void PixelProgressBar::UpdateLowStateAnimation(float delta_seconds) {
    blink_timer_ += delta_seconds;
    if (blink_timer_ >= AnimationConfig::BlinkInterval) {
        blink_timer_ = 0.0f;
        blink_on_ = !blink_on_;
    }
    // 根据 blink_on_ 切换 fill_color_
}
```

|------|--------|------|------|------|

| NPC 名字 | 微软雅黑 | Noto Sans SC | Bold | 16px |
| 面板标题 | 微软雅黑 | Noto Sans SC | Bold | 16px |
| HUD 信息 | 微软雅黑 | Noto Sans SC | Regular | 12px |
| 快捷键 | Arial | - | Regular | 10px |

### D.3 图集元数据 (.atlas.json) 格式

每张 PNG 图集配套同名 `.atlas.json`，程序通过 `SpriteAssetManager::LoadAtlas()` 加载。

**核心字段**:
- `id`: 图集唯一 ID（英文小写下划线）
- `texture`: PNG 相对路径
- `width/height`: 图集像素尺寸
- `frames[]`: 所有帧定义（x, y, w, h, pivotX, pivotY, duration）
- `animations[]`: 动画序列（id, name, loop, speed, frames[]）

**帧 ID 全局唯一**: `{atlas_id}_{帧名}`，如 `player_main_idle_down_0`

**示例**: 见 `assets/sprites/characters/player_main.atlas.json`

### D.4 UI 页面配置 (.ui.json) 格式

每个 UI 面板对应一个 `.ui.json`，程序通过 `UiPageManager::LoadPagesFromDirectory()` 加载。

**核心字段**:
- `page_id`: 页面 ID
- `width/height`: 面板尺寸
- `modal`: 是否模态
- `elements[]`: 元素列表（每项含 id/type/位置/样式/回调）

**元素样式绑定**: `style.sprite_id` 引用图集中的帧 ID

**示例**: 见 `assets/ui/pages/inventory.ui.json`

### D.5 渲染管线集成

**渲染顺序**:
```
1. WorldRenderer        → 游戏场景（玩家/NPC/作物/地形）
2. SpriteBatchRenderer  → 批量精灵（瓦片/粒子/作物）（可选）
3. UiPageManager       → 所有打开的页面
4. PixelGameHud         → 底层 HUD（向后兼容）
```

**每帧调用**:
```cpp
// GameApp::Render()
world_renderer.Render(window, world_state);
page_manager.Render(window);    // UI 页面
pixel_hud_->Render(window);     // 兼容层（可逐步迁移）
```
