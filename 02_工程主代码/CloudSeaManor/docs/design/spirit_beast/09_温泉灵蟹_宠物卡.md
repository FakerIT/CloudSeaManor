# 09_温泉灵蟹_宠物卡

文档版本：v1.0 | 稀有度：★★☆☆☆ | 分类：场景限定稀有伙伴

## 基础信息
- 初始获得：山庄温泉区升级完成后，自然出现触发获取事件
- 进化链：幼蟹 → 温泉蟹 → 赤焰灵蟹
- 推荐搭配NPC：秋棠
- 核心定位：灵气回复、体力续航、水火双系治愈型伙伴

## 世界观设定
山庄温泉地脉与火山灵气孕育的灵蟹，温泉区升级后被山庄灵气吸引而来；性格热情好客，爱热闹，会主动帮玩家打理温泉，是百年前山庄汤泉馆的守护灵，对秋棠的医术与汤泉配方有天生的亲近感。

## 个性与外观
### 基础个性
热情活泼，好客爱热闹，会在温泉里吐泡泡欢迎玩家，偏爱在温泉热水里待着，玩家泡温泉时会趴在玩家身边，主动帮玩家按摩，会用温泉热气给玩家准备小惊喜。
### 分阶外观
- 幼蟹：拳头大的浅红色幼蟹，蟹壳带热气纹路，双钳带微光，吐泡泡时会冒出热气。
- 温泉蟹：体型稍大，赤红色蟹壳，布满温泉热气纹路，双钳带赤光，移动时留下温热的水痕，吐泡泡时会生成温泉水雾。
- 赤焰灵蟹：终阶形态，通体赤红泛金辉，蟹壳布满灵脉纹路，双钳带熊熊灵火，周身环绕温泉热气，走过的地方会生成临时温泉泉眼。

## 成长与进化体系

|----------|----------|----------|

| 温泉蟹 | 羁绊3心 + 净化等级≥6级 + 汤泉工坊解锁 + 秋棠好感≥3心 | 全山庄灵气恢复加成、专属温泉配方解锁、水火技能强化 |

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【温泉滋养】被动技能
2心：解锁温泉区驻家功能、喂食互动、温泉体力恢复速度+30%，玩家灵气恢复+10%
3心：解锁成体进化、「灵蟹寻泉」支线剧情、专属温泉配方解锁
4心：解锁【全山庄灵气恢复】功能、雨天/温泉场景全能力+30%，秋棠制药/汤泉制作效率+25%
5心：解锁终阶进化、「赤焰镇地脉」隐藏剧情、终阶全屏灵气恢复技能、温泉区永久效果翻倍

### 好感获取加成
- 喂食喜好：温泉灵藻（+30）、火灵浆果（+20）
- 温泉互动：玩家每泡1次温泉，温泉灵蟹好感+5
- NPC联动：与秋棠互动时，温泉灵蟹额外获得12点好感/次

## 全技能详解
### 农活经营类
- 【灵泉滋养】（被动，常驻）：玩家在温泉区工作时，体力恢复速度+100%，灵气恢复速度+50%；与秋棠同场景时，汤泉配方制作速度+50%，药品品质+15%
- 【地热催生】（主动，冷却40秒）：用地热灵气催生周围农田作物，作物生长速度翻倍，对喜热作物有额外50%催生效果，温泉附近农田永久生效

### 战斗净化类
- 【热水冲击】（主动，冷却15秒）：释放热水冲击，范围内污染灵体污染值削减25%，被冲击的灵体进入烫伤状态，移动速度-50%，持续6秒
- 【灵火净化】（主动，冷却20秒）：释放赤焰灵火，单目标大幅削减40%污染值，对水系/冰系污染灵体有额外60%净化效率，命中后持续灼烧削减污染值
- 【赤焰灵泉】（终阶主动，冷却60秒）：生成全屏赤焰灵泉，所有污染灵体污染值持续大幅削减，玩家体力与灵气100%恢复，净化效率+100%，持续12秒

### 生活社交类
- 【全山庄灵气恢复】（被动，4心解锁）：每日凌晨，温泉灵蟹会释放灵气，恢复全山庄所有玩家与NPC的灵气，恢复量按温泉区等级增幅，最高50%
- 【专属温泉配方】（被动，3心解锁）：解锁专属「灵蟹温泉汤」配方，泡汤后玩家全天体力消耗-30%，净化效率+20%，NPC好感获取+15%
- 【温泉守护】（驻家被动）：驻留温泉区时，温泉区随机事件触发概率+30%，温泉效果永久翻倍，来访NPC泡汤后好感额外+10%

### 特殊云海类
- 【雾中灵泉】（被动）：浓云海天气时，全技能效果+30%，温泉灵气恢复效果翻倍；大潮天气时，可触发全山庄温泉灵气爆发，玩家全天灵气不衰减

## 战斗定位与用法
- **战斗定位**：治疗回复（灵气续航） + 水火双系 + 持久战续航
- **战斗风格**：以灵气回复保障队伍持续作战，热水冲击减速+灵火净化高输出
- **核心优势**：全山庄灵气恢复是永久性团队增益，赤焰灵泉终结技能提供全面恢复
- **天气优势**：浓云海+30%，大潮全天灵气不衰减
- **配合要点**：与所有阵容都是完美辅助，尤其与高消耗输出（界影狼、星尘鹿）和控场（云梦蝶）配合

## 战斗连招与搭配思路
- **连招思路**：热水冲击起手减速 → 灵火净化高输出 → 赤焰灵泉终结全恢复
- **属性克制**：对水系/冰系污染灵体净化效率+60%
- **阵容搭配**：温泉灵蟹 + 界影狼 + 岩龟 = 灵气续航+影刃爆发+嘲讽承伤
- **持久战搭配**：温泉灵蟹 + 灵芝精 + 玉蝶 = 灵气续航+持续治疗+群体减速

## 专属事件
1. 【温泉的小客人】：温泉区升级完成后触发，初始获取事件
2. 【灵蟹寻泉】：3心解锁，帮助温泉灵蟹找回地下温泉的核心泉眼，解锁成体进化
3. 【泡泡惊喜】：随机触发，温泉灵蟹吐泡泡给玩家带来温泉材料/小礼物，好感+15，获得专属汤泉配方
4. 【灵蟹护地脉】：5心终阶剧情，大潮天气时，温泉灵蟹镇压温泉地脉波动，守护山庄，解锁终阶进化

## 适配玩法场景
- 核心场景：温泉区经营、汤泉/药品制作、体力续航恢复、灵境持久战净化
- 适配阶段：全周期，尤其中期温泉区扩张、灵境深度探索续航期
