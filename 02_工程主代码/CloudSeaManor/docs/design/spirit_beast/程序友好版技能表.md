---

## 一、技能数据结构

每条技能保留以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| skill_id | string | 技能唯一标识（格式：`spiritId_skillName`）|
| name | string | 技能中文名 |
| type | enum | 技能类型：`passive`（被动）/ `active`（主动）/ `ultimate`（终阶）/ `home`（驻家）|
| target | enum | 目标类型：`self` / `single` / `area` / `all` / `global` |
| effect | string | 效果描述（用于程序实现） |
| cooldown | int | 冷却时间（秒），被动/驻家填0 |
| duration | int | 持续时间（秒），无持续填0 |
| weather_bonus | object | 天气加成，格式：`{"clear":0,"mist":0.2,"thick":0.35,"tide":0.5}` |
| unlock_heart | int | 解锁所需羁绊等级（1-5） |
| unlock_stage | enum | 解锁所需进化阶段：`infant`（幼体）/ `adult`（成体）/ `ultimate`（终阶）|

## 五、净化效率计算公式

```cpp
// 最终净化效率 = 基础效率 × (1 + 羁绊加成 + 天气加成 + 技能加成)
// 羁绊加成：1心+0%，2心+10%，3心+20%，4心+35%，5心+50%
// 天气加成：见各技能weather_bonus字段
// 技能加成：玩家装备/buff提供

float CalculatePurifyEfficiency(SpiritBeast& beast, const Skill& skill, Weather weather) {
    float baseEff = skill.baseEfficiency;
    float heartBonus = beast.heartLevel * 0.10f + (beast.heartLevel >= 3 ? 0.10f : 0.0f);
    float weatherBonus = skill.weatherBonus[weather];
    float skillBonus = skill.additionalBonus; // 部分技能自带加成
    return baseEff * (1.0f + heartBonus + weatherBonus + skillBonus);
}
```

## 六、污染值削减计算

```cpp
// 污染值削减量 = 基础削减 × 净化效率
// 精英灵体：基础污染值200%，净化难度×1.5
// BOSS灵体：基础污染值300%，净化难度×2.0

float CalculatePollutionReduction(float baseReduction, float purifyEfficiency, SpiritType type) {
    float difficultyMod = 1.0f;
    if (type == SpiritType::Elite) difficultyMod = 1.5f;
    if (type == SpiritType::Boss) difficultyMod = 2.0f;
    return baseReduction * purifyEfficiency / difficultyMod;
}
```
