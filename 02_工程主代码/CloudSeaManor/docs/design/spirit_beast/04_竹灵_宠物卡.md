# 04_竹灵_宠物卡

文档版本：v1.0 | 稀有度：★☆☆☆☆ | 分类：常驻普通伙伴

## 基础信息
- 初始获得：春季竹林种植「竹灵芽」后，成熟触发获取事件
- 进化链：幼灵 → 竹甲灵 → 翠皇灵
- 推荐搭配NPC：素锦、烟萝
- 核心定位：种植专精、群体禁锢、竹阵控场型伙伴

## 世界观设定
山庄后山百年竹林孕育的木系灵体，天生擅长种植与建造，是百年前山庄工匠的伙伴；性格勤劳害羞，不爱说话，只会默默帮玩家打理农田与家园，对素锦的织物、烟萝的花艺有天生的喜爱。

## 个性与外观
### 基础个性
勤劳内敛，害羞怕生，只会用竹叶晃动表达情绪，永远在默默干活，玩家靠近时会躲进竹篮里，熟悉后会主动帮玩家整理背包里的种植材料。
### 分阶外观
- 幼灵：巴掌大的竹叶精灵，头顶嫩芽，背着迷你小竹篮，身体半透明翠绿色。
- 竹甲灵：体型稍大，身体覆盖竹片铠甲，背着大竹篮，手持小竹锄，竹叶纹路清晰。
- 翠皇灵：终阶形态，通体翠绿泛玉光，头顶竹冠，身后竹翼，周身环绕竹灵，可自由操控竹枝生长。

## 成长与进化体系
| 进化阶位 | 解锁条件 | 核心成长 |
|----------|----------|----------|
| 幼灵 | 春季竹林种植竹灵芽成熟后触发 | 基础种植加速、自动施肥功能 |
| 竹甲灵 | 羁绊3心 + 净化等级≥5级 + 工坊解锁 + 素锦好感≥2心 | 种植效率翻倍、家具制作功能解锁、家园布置效率提升 |
| 翠皇灵 | 羁绊5心 + 净化等级≥15级 + 山庄守护值≥8000 + 完成「竹灵护山」终阶剧情 | 全地图灵植催生、终阶竹阵技能、素锦/烟萝羁绊共鸣翻倍 |

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【种植加速】被动技能
2心：解锁农田驻家功能、喂食互动、施肥效率+50%，作物生长速度+10%
3心：解锁成体进化、「竹灵寻根」支线剧情、【家具制作】技能
4心：解锁【家园布置增效】功能、春季/雨季全能力+30%、素锦/烟萝工坊制作效率+20%
5心：解锁终阶进化、「翠皇守竹林」隐藏剧情、全地图灵植催生技能、山庄建筑修复速度+15%

### 好感获取加成
- 喂食喜好：竹露清泉（+30）、灵竹嫩芽（+20）
- 种植操作：玩家每完成1次播种/浇水，竹灵好感+1
- NPC联动：与素锦/烟萝互动时，竹灵额外获得8点好感/次

## 全技能详解
### 农活经营类
- 【种植专精】（被动，常驻）：玩家种植操作速度+100%，自动为周围农田施肥，作物生长速度+20%；与素锦同场景时，作物高品质产出概率+15%
- 【竹灵催生】（主动，冷却30秒）：催生周围8格农田作物，直接前进1个生长阶段；春季雨季时，可催生全田作物，冷却缩减50%
- 【家具制作】（被动，3心解锁）：工坊制作家具时，制作速度+80%，材料消耗-20%，可解锁专属竹制高级家具配方

### 战斗净化类
- 【竹刺禁锢】（主动，冷却20秒）：在指定位置生成竹刺阵，禁锢范围内污染灵体，持续削减污染值，持续8秒；对地面类污染灵体有额外30%禁锢时长
- 【竹甲护盾】（主动，冷却25秒）：为玩家生成竹甲护盾，吸收40%的干扰伤害，护盾持续期间玩家体力不衰减，持续10秒
- 【翠皇竹阵】（终阶主动，冷却60秒）：生成全屏翠皇竹阵，所有污染灵体被永久禁锢，污染值持续大幅削减，玩家在阵内净化效率+80%，持续15秒

### 生活社交类
- 【家园布置增效】（被动，4心解锁）：玩家布置家园家具时，移动速度+50%，家具旋转/放置无冷却，可解锁专属竹林家园场景
- 【竹林守护】（驻家被动）：驻留竹林/农田时，作物枯萎概率降为0，野外竹子采集效率+100%，稀有竹材产出概率+20%
- 【自动施肥】（被动，常驻）：自动为周围农田施肥，无需玩家操作，施肥效率+50%

### 特殊云海类
- 【雾中竹灵】（被动）：薄雾天气时，全技能效果+25%，作物生长速度额外+15%；浓云海天气时，竹阵禁锢效果翻倍

## 战斗定位与用法
- **战斗定位**：控场禁锢 + 护盾辅助 + 种植专精
- **战斗风格**：以竹刺阵禁锢敌人，竹甲护盾保护队友，翠皇竹阵终结全场
- **核心优势**：群体禁锢是群怪战的核心控制，竹甲护盾提供稳定的团队保护
- **天气优势**：薄雾+25%、浓云海禁锢翻倍
- **配合要点**：与高输出（界影狼、青鸾）配合形成"控场+输出"的经典组合；与治疗（灵芝精）配合强化团队生存

## 战斗连招与搭配思路
- **连招思路**：竹刺禁锢起手控场 → 竹甲护盾保护 → 翠皇竹阵终结禁锢净化
- **属性克制**：对地面类污染灵体禁锢时长+30%（浓云海+60%）
- **阵容搭配**：竹灵 + 界影狼 + 灵芝精 = 竹刺禁锢+影刃爆发+灵芝治疗
- **群怪搭配**：竹灵 + 云梦蝶 + 玉蝶 = 竹阵禁锢+幻梦迷惑+灵粉减速

## 专属事件
1. 【竹灵发芽】：春季种植竹灵芽成熟后触发，初始获取事件
2. 【竹灵寻根】：3心解锁，帮助竹灵找回百年前的竹灵母根，解锁成体进化
3. 【害羞的小竹灵】：随机触发，玩家靠近时竹灵躲起来，互动后好感+15，获得稀有竹材
4. 【竹灵护山】：5心终阶剧情，暴雨天气时，竹灵守护后山竹林与山庄，解锁终阶进化

## 适配玩法场景
- 核心场景：大规模农田种植、工坊家具制作、家园布置装修、雨季日常经营
- 适配阶段：全周期，尤其新手开荒期、家园建设扩张期
