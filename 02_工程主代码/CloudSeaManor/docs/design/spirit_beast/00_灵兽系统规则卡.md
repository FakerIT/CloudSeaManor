# 00_灵兽系统规则卡

> 文档版本：v1.0 | 分类：核心系统规则 | 适用阶段：全周期

---

## 一、世界观定位

灵兽（Spirit Beast）是云海山庄的核心伙伴系统，与山庄、云海、灵脉深度绑定。每只灵兽既是玩家的战斗搭档，也是日常生活与经营的得力助手。

- **总数**：20种（含12种常驻+8种季节/事件限定）
- **进化**：全部支持三阶段进化（幼体 → 成体 → 终阶）
- **繁殖**：成年灵兽 + 灵界莲 ×1 + 云海浓度达标
- **携带上限**：6只（灵兽园高阶后可突破至9只）
- **功能定位**：农活辅助、战斗净化、生活陪伴、事件触发

## 二、出战与驻场规则

### 2.1 出战规则

| 规则 | 说明 |
|------|------|
| 同时出战上限 | 最多3只灵兽同时出战 |
| 切换机制 | 可随时在灵兽园切换出战/驻场灵兽 |
| 休息机制 | 战斗后灵兽进入30分钟休息期，期间无法再次出战（可用灵茶加速恢复） |
| 经验分配 | 战斗中所有出战灵兽平均分配净化经验 |
| 羁绊加成 | 出战灵兽与对应NPC同场景时，羁绊获取速度+30% |

### 2.2 驻场规则

| 规则 | 说明 |
|------|------|
| 驻场位置 | 每只灵兽有固定驻场区域（农田/茶园/矿洞/温泉/观云台等） |
| 驻场增益 | 驻场灵兽提供被动加成，无需出战即可生效 |
| 驻场切换 | 可随时将驻场灵兽换为出战位 |
| 灵兽园 | 灵兽园等级决定可驻场灵兽数量（Lv1: 3只, Lv2: 5只, Lv3: 8只） |

## 三、羁绊与好感系统

### 3.1 好感等级（1心～5心）

| 等级 | 解锁内容 |
|------|----------|
| 1心 | 跟随功能、基础互动、1个被动技能 |
| 2心 | 驻家功能、喂食互动、被动技能强化 |
| 3心 | 成体进化、专属剧情、主动技能解锁 |
| 4心 | 全地图功能、驻家被动强化、天气加成生效 |
| 5心 | 终阶进化、终阶隐藏剧情、终阶技能解锁 |

### 3.2 好感获取途径

| 途径 | 好感值 |
|------|--------|
| 喂食喜好食物 | +20～+30 |
| 完成专属工作（种田/采茶/挖矿等） | +1～+5/次 |
| NPC联动互动 | +8～+15/次 |
| 战斗胜利 | +10 |
| 完成专属事件 | +15～+30 |
| 进化成功 | +50 |

## 四、进化体系

### 4.1 三阶段进化

| 阶段 | 名称 | 条件 | 提升 |
|------|------|------|------|
| 幼体 | 初始形态 | 完成获取事件 | 基础被动技能 |
| 成体 | 第二形态 | 羁绊3心 + 净化等级门槛 + 特定设施 + NPC好感 | 主动技能解锁、被动强化、驻家功能 |
| 终阶 | 最终形态 | 羁绊5心 + 净化等级满 + 守护值门槛 + 终阶剧情 | 终阶技能解锁、终极被动、全属性大幅提升 |

### 4.2 进化材料

- 进化石（各系对应属性）
- 灵界莲（进化必备）
- 云海露（提升成功率）
- 专属材料（各灵兽不同，如古藤果/岩龟甲/云狐尾等）

## 五、技能分类

所有灵兽技能分为四类：**农活经营类 / 战斗净化类 / 生活社交类 / 云海专属类**

### 5.1 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 被动-常驻 | 获取后永久生效，无需操作 | 采茶速度+80%、温泉体力恢复 |
| 被动-条件 | 满足特定条件时生效 | 薄雾天气技能效果+25% |
| 主动-基础 | 1心解锁，可手动释放 | 风刃净化、岩甲护盾 |
| 主动-强化 | 3心解锁，成体形态可用 | 竹灵催生、影分身突袭 |
| 终阶-领域 | 5心解锁，终阶形态专属 | 太初云海域、界皇暗影域 |
| 驻家-被动 | 驻留特定区域时自动生效 | 观云台云海预报+100% |

### 5.2 技能冷却说明

- 战斗技能冷却：**8秒～25秒**（基础/强化）
- 终阶技能冷却：**60秒～90秒**
- 农活技能冷却：**30秒～60秒**
- 部分技能受云海天气影响，冷却可缩减50%

## 六、战斗净化系统

> **核心原则**：无死亡、无逃跑、无负面惩罚。战斗以净化为主，通过削减污染值使灵体恢复清明，而非消灭。

### 6.1 战斗定位

| 定位 | 说明 | 代表灵兽 |
|------|------|----------|
| 净化输出 | 快速削减污染值，优先击杀精英/BOSS | 云狐、太初雏龙 |
| 控场禁锢 | 限制灵体移动，群体控制 | 竹灵、古藤猿 |
| 护盾防御 | 吸收伤害，保护玩家和队伍 | 岩龟 |
| 治疗回复 | 恢复玩家体力灵气，提供续航 | 温泉灵蟹、雾鹿 |
| 辅助增益 | 预警、加速、减耗 | 观云雀、青鸾 |
| 天气特化 | 特定天气下效果翻倍 | 霜魂兔（雪天）、太初雏龙（大潮） |

### 6.2 污染值系统

- **初始污染值**：普通灵体 100%，精英 200%，BOSS 300%
- **净化目标**：将污染值削减至0%，灵体恢复清明
- **净化效率**：受灵兽羁绊等级、天气、技能加成影响
- **无死亡惩罚**：玩家净化失败，灵体暂时退散，不受负面惩罚

### 6.3 战斗流程

```
进入战斗 → 污染灵体出现 → 玩家选择出战灵兽（最多3只）
→ 释放技能削减污染值 → 达到100%净化 → 战斗胜利 → 获得经验与材料
→ 未完成净化 → 灵体暂时退散 → 可再次挑战 → 无任何惩罚
```

### 6.4 战斗场景适配

| 场景 | 特点 | 推荐定位 |
|------|------|----------|
| 日常清怪 | 大量低污染值灵体，速战速决 | 净化输出 + 控场 |
| BOSS战 | 高污染值精英/BOSS，持久消耗 | 护盾 + 治疗 + 净化输出 |
| 群怪净化 | 中等数量中等污染值 | 控场 + 辅助增益 |
| 灵境探索 | 地形复杂，需要快速通过 | 辅助预警 + 净化输出 |
| 夜间守护 | 持续骚扰，需要长期驻守 | 护盾 + 治疗 |

## 八、战斗惩罚机制

> **无死亡、无逃跑、无负面** — 这是云海山庄战斗系统的核心理念。

### 8.1 无死亡机制

- 战斗中不存在"击杀"概念，只有"净化完成"
- 未能完全净化的灵体将暂时退散，3分钟后重新出现
- 玩家灵兽在战斗中不会受伤或死亡
- 净化失败无任何惩罚，不掉装备、不扣资源

### 8.2 无逃跑机制

- 玩家可随时主动退出战斗，灵体自动进入休息状态
- 战斗中不可逃跑，但可以切换灵兽
- 退出战斗后，所有灵兽立即进入休息期（30分钟）

### 8.3 无负面机制

- 战斗失败：无任何惩罚
- 净化超时：灵体暂时退散，可稍后重新挑战
- 灵兽疲劳：通过灵茶/温泉/休息恢复，无永久负面效果
- 羁绊惩罚：无任何降低羁绊的机制（不存在）

## 九、隐藏彩蛋与终局

- **12灵兽收集**：集齐12种常驻灵兽并完成进化，解锁"小灵兽大典"庆典
- **20灵兽全收集**：集齐全20种灵兽，触发"灵兽大典"全村庆典
- **太初雏龙终阶进化**：完成真结局+全收集+终阶进化，触发"太初圣龙守护山庄"隐藏真结局
- **灵兽大比**：夏季灵兽赛、秋季灵兽大比节日活动中，灵兽可参赛展示
- **灵兽变异**：特定天气+高羁绊条件下，灵兽有概率出现稀有外观变异

## 十、与经营系统的深度绑定

- 灵兽的所有技能同时服务于**农活经营**与**战斗净化**两条线
- 战斗净化可提升山庄守护值，守护值影响建筑等级上限
- 灵兽羁绊提升可增强对应NPC的互动效率与好感获取
- 净化等级与经营等级共享经验池
- 灵兽驻场可触发专属经营事件（如茶灵猫的"茶园惊喜"、古藤猿的"建筑维修"）
