# 08_霜魂兔_宠物卡

文档版本：v1.0 | 稀有度：★★★☆☆ | 分类：季节限定珍稀伙伴

## 基础信息
- 初始获得：冬季雪天浓云海天气，雪山捕捉触发获取事件
- 进化链：幼兔 → 霜绒兔 → 月魂玉兔
- 推荐搭配NPC：烟萝、晚星
- 核心定位：群体减速、冰霜控场、冬季治愈陪伴型伙伴

## 世界观设定
雪山寒霜与月辉孕育的灵兔，只在冬季雪天浓云海时现身，百年前曾是山庄花艺师的伙伴；性格浪漫文艺，偏爱雪天与月光，会在雪天给玩家带来惊喜，是冬季山庄的天然守护灵，对烟萝的花艺、晚星的月相占卜有天生的共鸣。

## 个性与外观
### 基础个性
浪漫温柔，安静黏人，偏爱在雪地里跳跃，在月光下睡觉，会偷偷给玩家的窗台放雪绒花，高好感时会一直跟在玩家身边，是最适合陪伴玩家的治愈系伙伴。
### 分阶外观
- 幼兔：巴掌大的雪白幼兔，长耳朵尖带霜花，足下踩出霜痕，眼睛如月光，毛绒绒软乎乎。
- 霜绒兔：体型修长，雪白厚绒泛月辉，长耳朵布满霜花纹路，足下霜花清晰，跳跃时留下霜花轨迹。
- 月魂玉兔：终阶形态，通体雪白泛月华流光，长耳朵带月纹，身后飘着月轮光效，周身环绕霜雾，走过的地方会开出雪绒花。

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【冬作专精】被动技能
2心：解锁温室驻家功能、喂食互动、冬季作物生长速度+20%，雪天体力消耗-30%
3心：解锁成体进化、「霜兔寻花」支线剧情、【雪夜观星】专属事件
4心：解锁【全季节耐寒加成】功能、雪天/浓云海天气全能力+35%，烟萝/晚星好感获取+20%
5心：解锁终阶进化、「月魂照山庄」隐藏剧情、终阶月魂领域技能、冬季全山庄灵气获取+30%

### 好感获取加成
- 喂食喜好：雪绒花蜜（+30）、月露灵果（+20）
- 冬作操作：玩家每完成1次冬季作物种植/收获，霜魂兔好感+2
- NPC联动：与烟萝/晚星互动时，霜魂兔额外获得10点好感/次

## 全技能详解
### 农活经营类
- 【冬作专精】（被动，常驻）：冬季作物生长速度+50%，温室作物高品质产出概率+20%；与烟萝同场景时，花卉生长速度+30%，冬季作物额外+20%
- 【霜露滋养】（主动，冷却30秒）：用霜露滋养周围作物，作物生长速度翻倍，冬季可直接催生作物成熟，雪天冷却缩减50%
- 【全季节耐寒】（被动，4心解锁）：全季节作物均可在雪地种植，无耐寒惩罚，非冬季雪天作物也可正常生长

### 战斗净化类
- 【冰霜减速】（主动，冷却18秒）：释放冰霜领域，范围内所有污染灵体移动速度-70%，攻击间隔+80%，持续8秒；雪天持续时间+4秒，减速效果翻倍
- 【霜刃净化】（主动，冷却12秒）：释放霜刃，单目标削减30%污染值，对火系污染灵体有额外60%净化效率，命中后冻结目标3秒
- 【月魂霜域】（终阶主动，冷却60秒）：释放全屏月魂霜域，所有污染灵体被冻结定身，污染值持续大幅削减，玩家净化效率+80%，免疫所有干扰效果，持续15秒

### 生活社交类
- 【雪夜观星】（专属事件，3心解锁）：高好感时，雪夜触发观云台看雪专属事件，玩家与霜魂兔、对应NPC一起看雪观星，大幅提升NPC好感与羁绊值
- 【温室守护】（驻家被动）：驻留温室时，温室温度恒定，作物生长速度+30%，花卉异变概率+20%
- 【陪伴治愈】（被动，常驻）：跟随玩家时，玩家体力恢复速度+15%，每日可触发1次"治愈互动"，恢复玩家50%体力

### 特殊云海类
- 【雪雾霜灵】（被动）：雪天浓云海天气时，全技能效果+40%，玩家体力消耗-50%，冬季作物生长速度额外+30%；满月夜效果翻倍

## 战斗定位与用法
- **战斗定位**：控场减速（群体）+ 冰霜冻结 + 冬季治愈陪伴
- **战斗风格**：以冰霜减速冻结战场，霜刃净化精准输出，是雪天/冬季战斗的核心
- **核心优势**：群体减速+冻结是持久战和多目标战斗的神技，雪天效果翻倍
- **天气优势**：雪天浓云海+40%，满月夜翻倍
- **配合要点**：与高输出（界影狼、雷雀）配合最大化减速收益；与治疗（灵芝精）配合强化团队生存

## 战斗连招与搭配思路
- **连招思路**：冰霜减速先手控场 → 霜刃净化精准输出 → 月魂霜域终结冻结
- **属性克制**：对火系污染灵体净化效率+60%（雪天+90%）
- **阵容搭配**：霜魂兔 + 界影狼 + 灵芝精 = 冰霜减速+影刃爆发+灵芝治疗
- **雪天特化**：霜魂兔 + 温泉灵蟹 + 灵芝精 = 冰霜减速+灵气续航+持续治疗

## 专属事件
1. 【雪山的霜兔】：冬季雪天浓云海触发，初始获取事件
2. 【霜兔寻花】：3心解锁，帮助霜魂兔找回雪山之巅的雪绒花母株，解锁成体进化
3. 【窗台的雪绒花】：随机触发，清晨玩家窗台出现霜魂兔放的雪绒花，好感+15，获得稀有花艺材料
4. 【月魂照山庄】：5心终阶剧情，冬季满月夜，霜魂兔引月辉守护山庄，解锁终阶进化

## 适配玩法场景
- 核心场景：冬季种植经营、温室花艺培育、雪天灵境探索、污染灵体控场净化
- 适配阶段：冬季专属、中期温室扩张期、灵境控场辅助
