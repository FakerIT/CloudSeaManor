# 13_玉蝶_宠物卡

文档版本：v1.0 | 稀有度：★★☆☆ | 分类：常驻普通伙伴

## 基础信息
- 初始获得：花田种植「灵玉花」后，开放花田第5日后随机出现，互动获取
- 进化链：幼蝶 → 玉翼蝶 → 流光仙蝶
- 推荐搭配NPC：烟萝、素锦
- 核心定位：加速增益、辅助净化、花田守护型伙伴

## 世界观设定
花田灵玉花中孕育的灵蝶，翅膀由云海结晶凝成，在阳光下折射七彩光芒，是百年前山庄花艺师的挚友；性格温柔安静，不爱打扰人，但会在玩家不注意时悄悄帮忙打理花田，翅膀扇动时会洒落灵粉，让周围的花朵更加鲜艳。

## 个性与外观
### 基础个性
温柔内敛，安静害羞，喜欢在花田里飞舞，玩家靠近时会绕着人转圈表达亲昵，不喜欢嘈杂的环境，高好感时会轻轻落在玩家肩头休息。
### 分阶外观
- 幼蝶：指甲盖大小的透明玉色幼蝶，翅膀带细微光纹，飞舞时洒落微弱灵粉。
- 玉翼蝶：展开翅膀约手掌大小，通体玉白泛柔光，翅膀布满花瓣纹路，飞舞时身后留下七彩光带。
- 流光仙蝶：终阶形态，翼展宽大如屏，通体流光溢彩，翅膀由七彩云海结晶构成，飞舞时如流光瀑布洒落，周身环绕花灵。

## 成长与进化体系
| 进化阶位 | 解锁条件 | 核心成长 |
|----------|----------|----------|
| 幼蝶 | 花田开放第5日后随机出现，互动获取 | 基础灵粉加速、花田装饰功能 |
| 玉翼蝶 | 羁绊3心 + 净化等级≥5级 + 温室解锁 + 烟萝好感≥2心 | 花田产量提升、灵粉加速强化、加速增益技能解锁 |
| 流光仙蝶 | 羁绊5心 + 净化等级≥15级 + 山庄守护值≥8000 + 完成「流光护花田」终阶剧情 | 全图加速增益、终阶流光领域技能、烟萝/素锦羁绊共鸣翻倍 |

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【灵粉加速】被动技能
2心：解锁花田驻家功能、喂食互动、花卉生长速度+20%，灵粉产出+30%
3心：解锁成体进化、「玉蝶寻瓣」支线剧情、【群体加速】技能解锁
4心：解锁【全花田守护】功能、薄雾/浓云海天气全能力+30%，烟萝/素锦好感获取+20%
5心：解锁终阶进化、「流光映花田」隐藏剧情、终阶全图加速技能、全季节花卉产量+30%

### 好感获取加成
- 喂食喜好：灵玉花蜜（+30）、晨露花蜜（+20）
- 花田操作：玩家每完成1次花卉浇水/施肥，玉蝶好感+1
- NPC联动：与烟萝/素锦互动时，玉蝶额外获得8点好感/次

## 全技能详解
### 农活经营类
- 【灵粉加速】（被动，常驻）：玉蝶飞舞时，周围农田/花田作物生长速度+25%，灵粉洒落可提升附近灵兽恢复速度；与烟萝同场景时，花卉异变概率+15%
- 【花田守护】（被动，4心解锁）：驻留花田时，虫害概率降为0，花卉高品质产出概率+25%，稀有灵花出现概率+20%
- 【灵粉催生】（主动，冷却35秒）：释放大量灵粉，催生周围农田/花田作物前进1个生长阶段，花卉可直接进入盛开状态

### 战斗净化类
- 【灵粉减速】（主动，冷却18秒）：释放灵粉云雾，范围内所有污染灵体移动速度-60%，攻击间隔+60%，持续8秒；薄雾天气持续时间+3秒
- 【光蝶净化】（主动，冷却12秒）：释放流光蝴蝶，单目标削减25%污染值，对植物类污染灵体有额外40%净化效率，命中后目标进入减速状态
- 【流光仙域】（终阶主动，冷却65秒）：释放全屏流光仙域，所有污染灵体被减速80%，污染值持续大幅削减，玩家与全队伍净化效率+80%，持续12秒

### 生活社交类
- 【群体加速】（被动，3心解锁）：战斗开始时，为全队伍施加加速状态，全员移动速度+30%，持续到战斗结束；每日可触发3次
- 【花间信使】（被动，常驻）：可帮玩家传递礼物给NPC，效果+15%，附带灵粉装饰，NPC好感获取额外+10%
- 【花田守护】（驻家被动）：驻留花田时，作物枯萎概率降为0，花卉产出+20%，稀有灵花产出概率+15%

### 特殊云海类
- 【雾中流光】（被动）：薄雾天气时，全技能效果+25%，灵粉催生效果翻倍；浓云海天气时，减速效果持续时间+4秒

## 战斗定位与用法
- **战斗定位**：辅助增益 + 控场减速
- **战斗风格**：以灵粉减速控制战场，配合队友净化输出
- **核心优势**：群体加速和减速覆盖全队伍，是持久战和多目标战斗的最佳辅助
- **配合要点**：与净化输出型灵兽（云狐、界影狼）配合，可最大化减速收益

## 专属事件
1. 【花田的小精灵】：花田开放第5日后随机触发，初始获取事件
2. 【玉蝶寻瓣】：3心解锁，帮助玉蝶找回散落在花田的七彩花瓣，解锁成体进化
3. 【灵粉惊喜】：随机触发，玉蝶给玩家洒落大量灵粉，好感+15，获得灵玉花种子
4. 【流光护花田】：5心终阶剧情，大潮天气时，玉蝶守护花田抵御污染侵袭，解锁终阶进化

## 适配玩法场景
- 核心场景：花田种植经营、持久战辅助增益、花卉异变提升、NPC社交送礼
- 适配阶段：全周期，尤其花田扩张期、温室建设期、多目标持久战斗
