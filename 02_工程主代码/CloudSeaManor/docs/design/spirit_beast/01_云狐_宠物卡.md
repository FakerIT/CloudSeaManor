# 01_云狐_宠物卡

文档版本：v1.0 | 稀有度：★☆☆☆☆ | 分类：常驻普通伙伴

## 基础信息
- 初始获得：春季浓云海救治受伤幼崽
- 进化链：幼狐 → 云影狐 → 九尾云狐
- 推荐搭配NPC：阿茶、小满
- 核心定位：敏捷输出、闪避反制、幻影分身型伙伴

## 世界观设定
云海山庄春季浓云海中偶尔可见的灵狐，毛色如云，尾如雾，是山庄最亲近人的灵兽伙伴；性格活泼好奇，偏撒娇，对一切新鲜事物都充满兴趣，喜欢追逐云雾玩耍。

## 个性与外观
### 基础个性
活泼好奇，偏撒娇，喜欢跟着玩家到处跑，会用毛茸茸的尾巴蹭玩家的腿表达亲昵，高好感时会主动帮玩家叼来小礼物，是最适合新手的治愈系伙伴。
### 分阶外观
- 幼狐：巴掌大的雪白幼狐，淡蓝云纹若隐若现，尾部带雾气，跑动时如云影掠过。
- 云影狐：体型修长优雅，雪白毛皮泛云纹，尾部分叉如云雾，九条云尾飘动，奔跑时留下多重幻影。
- 九尾云狐：终阶形态，通体雪白泛九彩云光，九尾如九道云海，奔跑时如穿梭云层，周身环绕云雾，是云海中最灵动的一道光。

|----------|----------|----------|

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【云雾闪避】被动技能
2心：解锁茶园驻家功能、喂食互动、采茶速度+40%，体力消耗-20%
3心：解锁成体进化、「云狐寻雾」支线剧情、【幻影分身】技能解锁
4心：解锁【全地图采集】功能、薄雾/浓云海天气全能力+35%，阿茶好感获取+20%
5心：解锁终阶进化、「九尾云归」隐藏剧情、终阶幻影领域、全云海状态闪避+50%

### 好感获取加成
- 喂食喜好：云雾灵茶（+30）、晨露花蜜（+20）
- 采茶操作：玩家每完成1次采茶，云狐好感+2
- NPC联动：与阿茶/小满互动时，云狐额外获得8点好感/次

## 全技能详解
### 农活经营类
- 【采茶助手】（被动，常驻）：玩家采茶时效率+80%，自动识别高品质灵茶芽；与阿茶同场景时，额外+20%效率，灵茶高品质概率+15%
- 【云雾采集】（被动，4心解锁）：自动采集全地图野外灵材，采集速度+100%，稀有灵材产出概率+20%
- 【幻影催生】（主动，冷却40秒）：用幻影催生周围作物，生长速度翻倍，浓云海天气可催生全田

### 战斗净化类
- 【云雾闪避】（被动，常驻）：战斗中永久闪避30%攻击，浓云海天气闪避率+40%，大潮天气闪避率+60%
- 【幻影分身】（主动，冷却15秒）：分出3个幻影迷惑污染灵体，所有幻影均攻击目标，每个削减15%污染值，命中后目标命中率-50%，持续6秒
- 【九尾云归】（终阶主动，冷却70秒）：释放九尾云域，所有污染灵体被迷雾笼罩，命中率降为0，污染值持续削减，玩家净化效率+100%，持续15秒

### 生活社交类
- 【撒娇礼物】（被动，常驻）：高好感时自动代送礼物给目标NPC，送礼效果+20%，阿茶好感获取效率+25%
- 【云雾驻家】（驻家被动）：驻留茶园时，虫害概率降为0，茶叶高品质产出+30%
- 【云海快报】（被动，3心解锁）：浓云海天气前30分钟自动预警，可提前安排采集计划

### 特殊云海类
- 【雾中云狐】（被动）：浓云海天气时，全技能效果+35%，闪避率额外+20%；大潮天气时，幻影分身可分出5个幻影

## 战斗定位与用法
- **战斗定位**：净化输出（闪避反击） + 敏捷辅助
- **战斗风格**：以高闪避率减少受到的伤害，幻影分身迷惑敌人并反击，适合持续消耗战
- **核心优势**：闪避反击机制让云狐在浓云海/大潮天气表现极佳，幻影分身可同时攻击多个目标
- **天气优势**：浓云海+35%、大潮+50%
- **配合要点**：与护盾型（岩龟）配合形成"闪避+护盾"的铁壁阵容；与辅助型（观云雀）配合最大化情报优势

## 战斗连招与搭配思路
- **连招思路**：幻影分身起手迷惑 → 云雾闪避持续反击 → 九尾云归终结
- **属性克制**：对飞行类污染灵体净化效率+40%（浓云海+60%）
- **阵容搭配**：云狐 + 岩龟 + 观云雀 = 闪避反击+护盾防御+情报预警
- **BOSS搭配**：云狐 + 岩龟 + 灵芝精 = 闪避反击+护盾承伤+持续治疗

## 专属事件
1. 【云海的小狐狸】：春季浓云海救治受伤幼崽触发，初始获取事件
2. 【云狐寻雾】：3心解锁，帮助云狐找回散落的云雾碎片，解锁成体进化
3. 【撒娇的云狐】：随机触发，云狐蹭腿撒娇，好感+15，获得云雾灵茶
4. 【九尾云归】：5心终阶剧情，大潮天气时，云狐九尾归位，解锁终阶进化

## 适配玩法场景
- 核心场景：日常采茶、浓云海天气战斗、闪避流玩法、春季经营
- 适配阶段：全周期，尤其新手开荒期、茶园扩张期、浓云海战斗期
