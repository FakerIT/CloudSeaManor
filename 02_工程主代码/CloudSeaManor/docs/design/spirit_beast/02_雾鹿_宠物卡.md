# 02_雾鹿_宠物卡

文档版本：v1.0 | 稀有度：★★☆☆☆ | 分类：常驻普通伙伴

## 基础信息
- 初始获得：夏季山道跟随雾川事件
- 进化链：幼鹿 → 雾角鹿 → 圣雾白鹿
- 推荐搭配NPC：雾川、锦时
- 核心定位：稳定辅助、运输加速、危险预警型伙伴

## 世界观设定
云海山庄夏季山道跟随灵雾孕育的灵鹿，角挂云雾，步履轻盈，是山庄运输与预警的守护者；性格温柔忠诚，偏稳定辅助，会在浓云海中为玩家开辟安全的道路，永不迷路。

## 个性与外观
### 基础个性
温柔忠诚，步履轻盈，偏稳定辅助，不喜欢冒险，喜欢在熟悉的道路上行走，会用鹿角轻轻蹭玩家表达关心，高好感时会为玩家开路护航。
### 分阶外观
- 幼鹿：半人高的灰蓝幼鹿，角芽稚嫩，角挂细碎云雾，步伐轻盈无声。
- 雾角鹿：体型修长，灰蓝毛皮泛云光，角如云雾凝聚，踏步处留下云雾足迹，行走如云中漫步。
- 圣雾白鹿：终阶形态，巨型白鹿，角化作完整云雾灵阵，通体云雾缭绕，踏步时云雾如路，可带玩家全山庄瞬移，是山庄的守护之鹿。

|----------|----------|----------|

| 圣雾白鹿 | 羁绊5心 + 净化等级≥16级 + 山庄守护值≥8500 + 完成「圣雾归山」终阶剧情 | 全山庄瞬移、终阶预警领域、雾川/锦时羁绊共鸣翻倍 |

## 羁绊系统
### 1-5心解锁内容
1心：解锁跟随功能、基础互动、【运输加速】被动技能
2心：解锁山道驻家功能、喂食互动、运输速度+50%，体力消耗-40%
3心：解锁成体进化、「雾鹿寻道」支线剧情、【危险预警】技能解锁
4心：解锁【瞬移运输】功能、浓云海/大潮天气全能力+40%，全地图移动速度+60%
5心：解锁终阶进化、「圣雾归山」隐藏剧情、终阶预警领域、全山庄瞬移无消耗

### 好感获取加成
- 喂食喜好：云雾灵草（+30）、灵山露（+20）
- 运输操作：玩家每完成1次运输任务，雾鹿好感+2
- NPC联动：与雾川/锦时互动时，雾鹿额外获得10点好感/次

## 全技能详解
### 农活经营类
- 【运输加速】（被动，常驻）：自动运输收获物到仓库，运输速度+100%，体力消耗降为0；与雾川同场景时，运输效率+50%
- 【瞬移运输】（被动，4心解锁）：可瞬移运输物资到仓库，无需跑图，浓云海天气瞬移距离翻倍
- 【云雾播种】（主动，冷却40秒）：释放云雾灵气，大面积播种效率+100%，播种精度+50%

### 战斗净化类
- 【雾中预警】（被动，常驻）：战斗开始前标记所有污染灵体位置，玩家闪避率+25%，浓云海天气预警范围翻倍
- 【雾鹿之径】（主动，冷却20秒）：开辟浓云海安全路径，所有队友移动速度+50%，免疫浓云海负面效果，持续10秒
- 【圣雾领域】（终阶主动，冷却70秒）：释放全屏圣雾领域，所有污染灵体视野降为0，攻击命中率-70%，玩家净化效率+100%，免疫所有干扰，持续15秒

### 生活社交类
- 【全山庄导航】（被动，3心解锁）：永久标记全地图最优路线，不会走错路，不触发野兽袭击，运输任务效率+40%
- 【危险感知】（被动，常驻）：感知前方危险，提前30秒预警，浓云海天气可感知隐藏资源点
- 【山道守护】（驻家被动）：驻留山道时，野兽袭击概率降为0，运输任务效率+30%

### 特殊云海类
- 【雾中圣鹿】（被动）：浓云海天气时，全技能效果+40%，瞬移距离翻倍，预警准确度100%；大潮天气时，雾鹿之径持续时间翻倍

## 战斗定位与用法
- **战斗定位**：辅助增益（预警导航） + 机动支援 + 生存辅助
- **战斗风格**：以预警和导航保障队伍安全，雾鹿之径可免疫浓云海负面效果，是探索和持久战的核心辅助
- **核心优势**：浓云海导航+危险预警，让队伍永远不会陷入被动
- **天气优势**：浓云海+40%、大潮+50%
- **配合要点**：与任何阵容都是良好辅助，尤其与高输出（界影狼）和控场（云梦蝶）配合，实现"预警+控场+输出"的完美链条

## 战斗连招与搭配思路
- **连招思路**：雾中预警先手侦查 → 雾鹿之径开辟安全区 → 圣雾领域终结控场
- **属性克制**：对植物类污染灵体预警准确度+40%
- **阵容搭配**：雾鹿 + 云梦蝶 + 星尘鹿 = 完美预警+迷惑控场+暴击输出
- **探索搭配**：雾鹿 + 风铃狐 + 观云雀 = 全地图导航+全队加速+情报预警

## 专属事件
1. 【山道的小鹿】：夏季山道跟随雾川事件触发，初始获取事件
2. 【雾鹿寻道】：3心解锁，帮助雾鹿找回浓云海中的古老路径，解锁成体进化
3. 【雾中的足迹】：随机触发，雾鹿在浓云海中为玩家开辟道路，好感+15，获得云雾灵草
4. 【圣雾归山】：5心终阶剧情，大潮天气时，雾鹿开辟全山庄安全路径，解锁终阶进化

## 适配玩法场景
- 核心场景：浓云海运输导航、全地图探索、日常运输任务、危险预警感知
- 适配阶段：全周期，尤其中期全地图扩张期、浓云海日常期、危险探索期
