## 一、净化机制详解

### 1.1 污染值系统

污染值（Pollution Value）是污染灵体的核心属性，从0%到100%（或更高）。

**污染值结构**：
- 100% = 灵体完全被污染，呈现暗黑色
- 75% = 灵体呈现灰黑色，仍有微弱光点
- 50% = 灵体呈现灰色，光点增多
- 25% = 灵体呈现浅灰色，轮廓变清晰
- 0% = 灵体恢复清明，净化完成

**污染值计算公式**：
```
最终污染值削减 = 技能基础净化量
                  × 净化效率
                  × 天气倍率
                  × (1 + 羁绊加成)
                  × 元素克制倍率
                  × (1 - 敌人抗性)

净化效率 = 基础净化效率 × 羁绊加成 × 天气加成 × Buff加成
羁绊加成 = 1 + 羁绊等级 × 10%  (5心时 = 1.5)
天气倍率 = 晴×1.0 / 薄雾×1.2 / 浓云海×1.35 / 大潮×1.5
元素克制 = 克制时×1.5 / 被克制时×0.75 / 中性×1.0
```

**示例计算**：
```
玩家使用界影狼（3心=+20%）的风刃净化
基础净化量：40%污染值
净化效率：1.0 × 1.2 × 1.35 = 1.62
天气：浓云海=×1.35
羁绊：+20%
最终净化量：40 × 1.0 × 1.35 × 1.2 = 64.8%污染值
```

## 二、能量系统详解

### 2.1 能量基础

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始能量 | 100点 | 战斗开始时的能量 |
| 最大能量 | 100点 | 上限 |
| 能量回复 | 5点/秒 | 每秒自然回复 |
| 最低消耗 | 0点 | 被动技能不消耗 |

### 2.2 能量消耗来源

| 来源 | 消耗量 | 说明 |
|------|--------|------|
| 主动技能 | 5~50点 | 各技能不同 |
| 灵体攻击（被命中） | 技能伤害值 | 被灵体攻击时 |
| DEBUFF叠加 | 0.5~2点/秒 | 部分DEBUFF持续消耗 |
| 地形效果 | 1~5点/秒 | 如毒雾区、泥沼等 |

### 2.3 能量回复来源

| 来源 | 回复量 | 说明 |
|------|--------|------|
| 自然回复 | 5点/秒 | 基础回复 |
| 被动回复 | +2~10点/秒 | 灵兽伙伴被动 |
| 治愈孢子 | 每5秒+5%最大能量 | 灵芝精被动 |
| 温泉滋养 | 每5秒+8%最大能量 | 玉泉鹿被动 |
| 技能回复 | 10~30点 | 技能消耗的能量 |

### 2.4 能量耗尽机制

当能量耗尽（0点）时：
- 玩家进入"虚弱状态"：所有主动技能无法释放
- 净化效率降低50%
- 被动技能不受影响
- 需要等待自然回复或使用治疗技能

## 三、敌人攻击机制详解

### 3.1 敌人攻击模式

**接近型**：
- 缓慢接近玩家
- 到达攻击范围后进行近战攻击
- 攻击后有短暂的停顿（攻击窗口）

**远程型**：
- 与玩家保持一定距离
- 周期性发射投射物
- 投射物可以被打掉

**场地型**：
- 固定在战场某处
- 通过场地效果攻击（如毒雾、藤蔓）
- 通常有召唤机制

**游走型**：
- 在战场上随机移动
- 偶尔发动突袭
- 难以被锁定

### 3.2 敌人攻击类型

| 攻击类型 | 伤害表现 | 应对方式 |
|---------|---------|---------|
| 近战打击 | 扇形/单体，直接命中 | 闪避、护盾 |
| 投射物 | 飞行道具，可被打掉 | 护盾、击落 |
| 持续AOE | 区域伤害，周期性 | 离开区域、治疗 |
| 地形陷阱 | 被动触发 | 绕开、解除 |
| 即时全屏 | 必定命中 | 护盾、减伤 |

### 3.3 干扰伤害机制

敌人攻击不会直接减少玩家HP，而是消耗能量：

```
实际能量消耗 = 基础伤害 × (1 - 护盾减免) × (1 - 减伤buff)
```

当能量耗尽时：
- 玩家进入"灵气枯竭"状态
- 净化效率-50%
- 移动速度-30%
- 需要30秒不受伤才能恢复

## 四、元素克制系统详解

### 4.1 五行相生相克

```
相生：金生水、水生木、木生火、火生土、土生金
相克：水克火、火克金、金克木、木克土、土克水
```

### 4.2 克制效果

| 克制关系 | 净化效率加成 | 敌人攻击效果 |
|---------|------------|------------|
| 完美克制（2级克制）| +75% | 被克制方攻击力-30% |
| 一般克制 | +50% | 被克制方攻击力-15% |
| 中性 | 0% | 正常 |
| 被克制 | -25% | 攻击效果+15% |

### 4.3 光暗特殊克制

|------|---------|---------|
| 光克暗 | +50% | 驱散暗系敌人所有Buff |
| 暗克光 | +50% | 隐身、瞬移更难被侦测 |
| 光暗中性 | 0% | 无特殊效果 |

### 4.4 元素实战应用

| 敌人元素 | 推荐克制灵兽 | 推荐克制技能 |

## 六、Buff系统详解

### 6.1 玩家可获得的Buff

| Buff名称 | 效果 | 持续时间 | 来源 |
|----------|------|----------|------|
| 净化增效 | 净化效率+20% | 15秒 | 观云雀·灵体预警 |
| 全队加速 | 移动速度+30% | 持续 | 玉蝶·群体加速 |
| 岩甲护盾 | 吸收伤害+40% | 10秒 | 岩龟·岩甲护盾 |
| 灵芝护盾 | 吸收30%HP伤害 | 12秒 | 灵芝精·灵芝护盾 |
| 群体治疗 | 恢复40%HP | 即时 | 灵芝精·群体治愈 |
| 星夜祝福 | 经验+30% | 夜间持续 | 星尘鹿·星夜祝福 |
| 群体加速 | 移动+30% | 持续 | 风铃狐·全队加速 |
| 疾风净化 | 移动+50% | 持续 | 风铃狐·风驰电掣 |
| 泉水治疗 | 每5秒+8%HP | 持续 | 玉泉鹿·泉水治疗 |
| 云海守护 | 无敌+灵气无限 | 10秒 | 太初雏龙·云海守护 |

### 6.2 Buff叠加规则

| Buff类型 | 叠加方式 | 最大层数 |
|----------|---------|---------|
| 净化增效 | 叠加数值 | 无上限 |
| 移动速度 | 叠加百分比 | 5层 |
| 护盾 | 独立叠加 | 3层 |
| 治疗 | 取最大值 | 1层 |
| 经验加成 | 叠加百分比 | 无上限 |

### 6.3 Buff优先级

当多个Buff作用于同一属性时：
1. 被动Buff始终生效
2. 主动Buff取最高值
3. 装备Buff叠加计算
4. 最终效果 = 基础值 × (1 + 被动%) × (主动MAX%) × (装备%)

## 七、战斗流程详解

### 7.1 典型战斗流程（普通灵体）

```
T+0s    战斗开始
         - 战场加载
         - 敌人出场动画
         - 天气效果应用
         - BUFF应用（如观云雀预警）

T+1s    敌人接近
         - 敌人开始移动
         - 玩家可开始释放技能

T+3s    第一次攻击
         - 敌人释放攻击
         - 玩家被命中
         - 能量消耗
         - DEBUFF应用

T+5s    第一次反击
         - 玩家使用技能
         - 净化量应用
         - 敌人DEBUFF

T+10s   净化完成
         - 敌人净化动画
         - 奖励发放
         - 战斗结束
```

### 7.2 BOSS战流程

```
T+0s    BOSS登场
         - 剧情对话
         - 阶段说明
         - 玩家准备时间（5秒）

T+5s    第一阶段开始
         - 正常攻击模式
         - 玩家熟悉BOSS机制

T+60s   第一阶段完成
         - BOSS虚弱动画
         - 进入第二阶段

T+120s  BOSS战结束
         - 最终净化动画
         - 剧情结算
         - 奖励发放
```

### 7.3 连续挑战流程

```
第一波（3只）→ 第二波（4只）→ 第三波（5只）→ BOSS登场
     ↓              ↓              ↓           ↓
  清理10秒      清理15秒      清理20秒    最终决战
```

## 八、战斗AI行为详解

### 8.1 敌人AI状态机

```
┌─────────┐
│  IDLE   │ 待机，观望
└────┬────┘
     │
     ▼ 玩家进入视野
┌─────────┐
│APPROACH │ 接近玩家
└────┬────┘
     │
     ▼ 到达攻击范围
┌─────────┐
│ ATTACK  │ 执行攻击
└────┬────┘
     │
     ├──→ 攻击完成 → COOLDOWN（冷却）→ APPROACH
     │
     ▼ 受到控制
┌─────────┐
│CONTROLLED│ 被控制（晕/冰/禁锢）
└────┬────┘
     │
     ▼ 控制结束
     APPROACH
```

### 8.2 精英/BOSS特殊AI

**精英AI额外行为**：
- 生命值低于50%后进入"狂暴状态"：攻击速度+50%，但防御-30%
- 周期性使用特殊技能（每30秒一次）
- 召唤小怪助战（每45秒一次）

**BOSS AI额外行为**：
- 多阶段机制：每个阶段有不同的行为模式
- 阶段转换：特定生命值触发
- 召唤联动：与战场机制互动（如潮汐BOSS召唤小潮灵）
- 狂暴机制：最后阶段攻击速度+100%

### 8.3 灵兽伙伴AI

```
每帧检查：
  1. 是否在战斗中？ → 否则跳过
  2. 是否有可用技能？ → 无可用则普攻
  3. 选择最高优先级的技能
     - 优先：治疗玩家（玩家能量<30%）
     - 其次：控制敌人（敌人攻击前摇）
     - 再次：输出伤害（净化）
  4. 选择目标
     - 优先：污染值最高的敌人
     - 其次：正在攻击玩家的敌人
     - 再次：最近的敌人
  5. 释放技能
```

**灵兽AI优先级**：
1. 紧急治疗（玩家能量<30%）
2. 紧急护盾（敌人即将攻击）
3. 控制敌人（敌人即将释放大招）
4. 净化输出（正常战斗）

| 事件 | 触发概率 | 效果 | 应对 |
|------|---------|------|------|

| 连携类型 | 条件 | 效果 |

- 查看敌人详细属性
- 规划下一轮战术
- 更换技能配置
- 调整站位

## 十二、战斗难度自适应

### 12.1 玩家等级与敌人等级匹配

| 玩家平均等级 | 敌人难度 | 敌人等级 |
|-------------|---------|---------|
| 1-5 | 简单 | 玩家等级-3 |
| 5-10 | 普通 | 玩家等级-2 |
| 10-18 | 困难 | 玩家等级±0 |
| 18-25 | 挑战 | 玩家等级+2 |
| 25-30 | 地狱 | 玩家等级+5 |
| 30+ | 传说 | 固定最高级 |

### 12.2 动态难度调整

如果玩家在战斗中持续受挫：
- 连续3次战斗失败后，下次敌人污染值-10%
- 能量耗尽后，下次战斗初始能量+20
- 超过5分钟未完成净化，敌人攻击速度-20%

如果玩家过于顺利：
- 连续3次战斗一次受击都没有，下次敌人+10%污染值
- 战斗时间<30秒，下次敌人+15%污染值

### 12.3 每日挑战
