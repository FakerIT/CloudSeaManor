## 一、功能对比总览

### 1.1 已实现功能（85%完成度）

| 类别 | 云海山庄 | 星露谷 | 差距 |
|------|---------|--------|------|
| 农业 | 30+作物，灵茶系统 | 60+作物 | 内容差距 |
| 畜牧 | 鸡舍/畜棚 | 6种动物+产物加工 | 持平 |
| 战斗 | 灵界净化战斗 | 矿井战斗 | 玩法差异 |
| 社交 | 13 NPC+结婚 | 30+NPC+12可结婚 | 内容差距 |
| 节日 | 12节日 | 27节日事件 | 持平 |
| 加工 | 工坊+制茶 | 果酒桶+奶酪压榨机 | 持平 |
| 成就 | 20+成就 | 100+成就 | 内容差距 |

### 1.2 缺失核心功能

| 功能 | 星露谷实现方式 | 云海山庄建议实现 | 难度 |
|------|---------------|-----------------|------|
| **钓鱼系统** | 独立钓鱼小游戏 | 云海钓鱼小游戏 | ⭐⭐⭐ |
| **电视机系统** | 每日提示+食谱+天气 | 观云镜/灵泉井 | ⭐⭐ |
| **Bundle收集** | 社区中心Bundles | 茶灵契约收集 | ⭐⭐ |
| **特殊区域** | 沙漠+姜岛+矿洞 | 茶境秘境+虚空裂隙 | ⭐⭐⭐ |
| **多人联机** | Steam P2P | 未来预留 | ⭐⭐⭐⭐⭐ |

## 二、推荐拓展功能详细设计

### 2.1 【高优先级】钓鱼系统 - 云海钓鱼

#### 2.1.1 设计理念

借鉴星露谷的"鱼饵+浮漂"钓鱼机制，设计适合云海山庄的"灵气+云钩"钓鱼系统。

#### 2.1.2 核心机制

```
┌─────────────────────────────────────────────────────────────┐
│                      云海钓鱼小游戏                          │
├─────────────────────────────────────────────────────────────┤
│  [难度条] ████████████░░░░░░░░  目标：使绿色区域对准鱼     │
│                                                             │
│  [浮漂] ═══════════════════════════  玩家控制              │
│  [鱼]      ●━━━━━━━━━━●           鱼会左右移动             │
│                                                             │
│  [操作]  空格键：抛竿/收竿  A/D：控制浮漂                   │
└─────────────────────────────────────────────────────────────┘
```

#### 2.1.3 鱼类设计

| 分类 | 鱼名 | 出现条件 | 售价 | 稀有度 |
|------|------|---------|------|--------|
| 普通鱼 | 云鲤 | 四季/普通水域 | 30g | ★ |
| 普通鱼 | 雾鳝 | 春/薄雾天 | 50g | ★ |
| 稀有鱼 | 潮汐锦鲤 | 大潮日 | 200g | ★★★ |
| 传说鱼 | 太初鲲 | 灵界深层 | 1000g | ★★★★★ |
| 事件鱼 | 月华鱼 | 中秋节 | 150g | ★★★ |

#### 2.1.4 实施计划

```
Phase 1: 数据结构
- 新增 FishingSystem.hpp/cpp
- 新增 FishTable.csv (鱼种数据)
- 新增 FishingRod 工具类型

Phase 2: 小游戏实现
- 钓鱼进度条UI
- 鱼AI移动逻辑
- 成功/失败判定

Phase 3: 产出接入
- 鱼类物品加入背包
- 鱼类可加工（烤鱼/鱼汤）
- 鱼类可赠礼
```

### 2.2 【高优先级】电视机/日历系统 - 观云镜

#### 2.2.1 设计理念

星露谷的电视机提供每日提示、食谱和天气预告。云海山庄可以用"观云镜"替代，位于主屋内。

#### 2.2.2 观云镜功能

```
┌─────────────────────────────────────────────────────────────┐
│                      ☁️ 观云镜 ☁️                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [每日运势]  "今日宜：种植 忌：远行"                        │
│                                                             │
│  [天气预报]  明日：薄雾  后日：晴  大潮日：第7天            │
│                                                             │
│  [今日食谱]  "灵茶食谱：云雾茶 - 灵茶+露水"                │
│                                                             │
│  [云海观测]  "今日云海浓度：65% 适合灵茶生长"              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 2.2.3 数据结构

```cpp
// assets/data/daily/fortune_tips.csv
id,season,weather,tip_text
fortune_001,spring,,今日宜播种，新芽将茁壮成长
fortune_002,summer,rain,雨润万物，适合整理茶园
fortune_003,autumn,,秋高气爽，适合探索后山
fortune_004,winter,,冬日宜静养，炉火旁品茶最佳

// assets/data/daily/recipe_tips.csv
id,season,recipe_id,unlock_condition
recipe_tip_001,spring,cloud_mist_tea,farming_level>=2
recipe_tip_002,summer,rain_brew,fishing_level>=1
```

#### 2.2.4 实施计划

```
Phase 1: UI实现
- 新增 MysticMirrorPanel 类
- 主屋增加交互点

Phase 2: 功能接入
- 每日运势随机抽取
```

```
│                                                             │

│                                                             │
```

### 2.5 【长期目标】多人联机 - 云海共修

#### 2.5.1 设计理念

星露谷支持4人合作。云海山庄未来可支持多人联机，采用"共修"概念——玩家共同经营庄园。

#### 2.5.2 多人模式设计

```
┌─────────────────────────────────────────────────────────────┐
│                      🌟 云海共修 🌟                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [模式1] 本地双人                                          │
│  - 键盘分键（玩家1：WASD，玩家2：方向键）                   │
│  - 分割屏幕显示                                            │
│                                                             │
│  [模式2] 远程联机（未来）                                  │
│  - Steam P2P 或 自建服务器                                 │
│  - 最多4人同屏                                            │
│                                                             │
│  [共修机制]                                                │
│  - 庄园共享：农田、设施共享                                 │
│  - 劳动分工：每人专注1-2个系统                             │
│  - 关系独立：NPC关系各自独立                                │
│  - 资源共享：背包共享，金币独立                             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 2.5.3 技术挑战

| 挑战 | 解决方案 | 难度 |
|------|---------|------|
| 同步 | 主机权威模式 | ⭐⭐⭐⭐ |

| 存档 | 主机存档+云同步 | ⭐⭐⭐⭐ |
| UI | 独立视口渲染 | ⭐⭐⭐⭐ |

## 四、与星露谷的差异化竞争

### 4.1 云海山庄独特优势

| 特色 | 星露谷对比 | 强化方向 |
|------|-----------|---------|
| **灵茶系统** | 无 | 扩展为完整茶文化体系 |
| **云海天气** | 无 | 天气对作物影响深化 |
| **大潮机制** | 无 | 周期性事件核心驱动 |
| **茶灵收集** | 无 | Bundle系统核心差异化 |
| **东方美学** | 西方农场 | 国风视觉+音乐 |
| **治愈氛围** | 略有 | 强化禅意元素 |

### 4.2 建议深挖功能

1. **茶道系统**：品茶Buff、客人接待、茶室装修
2. **云海经济学**：云海浓度影响物价、灵气交易
3. **灵兽进化**：收集灵气、进化形态
4. **剧情深度**：玄清背景故事、太初起源
