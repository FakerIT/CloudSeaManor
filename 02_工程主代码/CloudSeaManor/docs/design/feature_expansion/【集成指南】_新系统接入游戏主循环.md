# 新系统集成指南

> 版本：v1.0 | 日期：2026-05-02
> 适用范围：钓鱼系统、观云镜系统、茶灵契约系统

---

## 一、集成概述

本指南说明如何将以下三个新系统接入游戏主循环：
1. **钓鱼系统** (FishingSystem + FishingUI)
2. **观云镜系统** (MysticMirrorSystem + MysticMirrorUI)
3. **茶灵契约系统** (ContractSystem + ContractUI)

## 四、触发钓鱼的方法

### 4.1 方法一：按F键触发

```cpp
void GameApp::TryStartFishing() {
    // 检查玩家是否在水边
    if (IsPlayerNearWater()) {
        auto& fishing_ui = CloudSeamanor::engine::GetFishingUI();
        auto& fishing_system = CloudSeamanor::engine::GetFishingSystem();

        // 获取当前位置对应的钓鱼地点
        std::string location_id = GetCurrentFishingLocation();
        int difficulty = fishing_system.CalculateFinalDifficulty(1, location_id);

        // 开始钓鱼小游戏
        fishing_ui.StartFishing(difficulty);

        // 设置完成回调
        fishing_ui.SetOnCatchCallback([](bool success, const std::string& fish_id) {
            if (success) {
                // 钓鱼成功，将鱼添加到背包
                GetInventory().AddItem(fish_id, 1);
                SetHintMessage("钓到了 " + GetFishName(fish_id) + "！");
            }
        });
    } else {
        SetHintMessage("这里不能钓鱼，需要靠近水边。");
    }
}
```

### 4.2 方法二：按E键与钓鱼点交互

在 `PlayerInteractRuntime::HandlePrimaryInteraction()` 中添加：

```cpp
// 检查是否是钓鱼点
if (object.HasTag("fishing_spot")) {
    auto& fishing_ui = CloudSeamanor::engine::GetFishingUI();
    auto& fishing_system = CloudSeamanor::engine::GetFishingSystem();

    std::string location_id = object.GetProperty("location_id");
    int difficulty = fishing_system.CalculateFinalDifficulty(1, location_id);

    fishing_ui.StartFishing(difficulty);
    return true;
}
```

## 七、已知问题和限制

### 7.1 存档兼容性

当前实现中，新系统的状态存储需要与存档系统集成。请确保：
- 定义新系统的存档数据结构
- 在存档加载/保存时同步调用

### 7.2 音频集成

新系统目前没有音效反馈，后续需要添加：
- 钓鱼成功音效
- 观云镜打开音效
- 契约完成音效

### 7.3 性能考虑

新系统使用单例模式，无额外性能开销。
