### 1.1 工具升级面板 ✅

**文件**:
- `include/CloudSeamanor/engine/PixelToolUpgradePanel.hpp`
- `src/engine/PixelToolUpgradePanel.cpp`

**功能**:
- Tab切换（工具/洒水器/肥料）
- 7种工具展示
- 经验进度条
- 工具效果显示
- 已集成现有的ToolSystem

**集成方式**:
```cpp
// 在GameApp或面板管理器中
PixelToolUpgradePanel tool_panel;
tool_panel.SetFontRenderer(font_renderer);

// 更新数据
ToolUpgradeViewData data;
auto& tool_sys = ToolSystem::Instance();  // 现有系统
for (int i = 0; i < 7; ++i) {
    auto type = static_cast<ToolType>(i);
    data.tools[i].name = tool_sys.ToolTypeName(type);
    data.tools[i].tier_name = tool_sys.ToolTierName(tool_sys.GetTier(type));
    data.tools[i].exp_ratio = tool_sys.GetExp(type) / tool_sys.GetExpForNext(type);
}
tool_panel.UpdateData(data);
```

## 三、代码集成指南

### 3.1 工具工坊面板集成

```cpp
// 在GameApp.hpp中声明
#include "CloudSeamanor/engine/PixelToolUpgradePanel.hpp"
class PixelToolUpgradePanel m_tool_upgrade_panel;

// 在GameApp.cpp中初始化
void GameApp::InitializeUI() {
    m_tool_upgrade_panel.SetFontRenderer(m_font_renderer);
    m_tool_upgrade_panel.SetOnUpgradeCallback([this](int tool_index) {
        return TryUpgradeTool(static_cast<ToolType>(tool_index));
    });
}

// 在快捷键处理中
case Keyboard::T:
    if (!m_tool_upgrade_panel.IsOpen()) {
        m_tool_upgrade_panel.Open();
        UpdateToolUpgradePanelData();
    }
    break;
```

### 3.2 节日小游戏集成

```cpp
// 在节日事件中启动
void StartFestivalMiniGame(MiniGameType type) {
    m_festival_game_panel.StartGame(type);
    EnterGameplayState(GameState::MiniGame);
}

// 游戏循环中
void Update() {
    if (m_festival_game_panel.IsGameRunning()) {
        m_festival_game_panel.Update(delta_seconds);
    }
}

// 获取结果
if (m_festival_game_panel.IsGameEnded()) {
    auto result = m_festival_game_panel.GetResult();
    player.AddGold(result.reward_gold);
}
```
