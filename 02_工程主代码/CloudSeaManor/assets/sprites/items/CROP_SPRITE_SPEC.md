# 农作物精灵图集规范文档

> 版本：1.0 | 日期：2026-05-01 | 状态：待美术制作
>
> 本文档定义云海山庄所有农作物的精灵图集规范。

---

## 概述

### 统计信息
- **作物总数**: 30种（普通作物）
- **灵茶总数**: 20种
- **每种作物生长阶段**: 5个
- **每种作物精灵帧数**: 2-4帧动画
- **总精灵图数量**: 约250张

### 尺寸规范
- **单帧尺寸**: 16x16 像素
- **建议缩放**: 4x (渲染时显示为 64x64)
- **调色板限制**: 不超过32色

## 精灵帧规范

### 每个作物需要以下精灵帧

| 阶段 | 精灵ID格式 | 帧数 | 描述 |
|------|-------------|------|------|
| 种子 | `crop_xxx_seed` | 1帧 | 棕色小点 |
| 幼苗 | `crop_xxx_stage1` | 2帧 | 小绿芽 |
| 成长 | `crop_xxx_stage2` | 2帧 | 矮小植株 |
| 成熟 | `crop_xxx_stage3` | 2帧 | 开花/结果前 |
| 可收获 | `crop_xxx_stage4` | 2帧 | 果实成熟 |
| 高品质收获 | `crop_xxx_golden` | 2帧 | 金边光晕 |

### 示例命名

```
crop_turnip_seed.png      # 萝卜种子
crop_turnip_stage1.png    # 萝卜幼苗
crop_turnip_stage2.png    # 萝卜成长
crop_turnip_stage3.png    # 萝卜成熟
crop_turnip_stage4.png    # 萝卜可收获
crop_turnip_golden.png    # 萝卜金边收获
```
