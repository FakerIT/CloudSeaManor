# 灵界敌人精灵规范文档

> 文档版本: v1.0 | 更新日期: 2026-05-02 | 覆盖: 24种敌人精灵

---

## 通用规范

### 尺寸规格
- 所有敌人精灵统一使用 **64x64 像素** 基础尺寸
- 动画帧尺寸: 64x64 像素
- 图集布局: 横向排列，每帧64px
- 高清模式: 128x128 像素 (2x缩放)

### 文件命名
```
格式: enemy_[enemy_id]_[state].png
示例: enemy_cloud_wisp_idle.png
      enemy_flame_imp_attack.png
      enemy_void_lord_boss.png
```

### 状态分类
| 状态 | 帧数 | 说明 |
|------|------|------|
| idle | 4-6 | 待机/漂浮状态 |
| move | 4-6 | 移动/飘动 |
| attack | 4-6 | 攻击动作 |
| hurt | 2-4 | 受伤 |
| die | 4-6 | 消散/死亡 |
| skill | 6-8 | 特殊技能 |

### 色彩规范
- 主色调根据元素属性定义
- 污染灵体使用深紫/暗绿渐变
- 净化完成使用渐变消散效果

## 攻击特效规范

### 净化攻击 (Purify Attack)
- 颜色: 金色/白色渐变
- 粒子: 光芒粒子向外扩散
- 音效: 清脆的铃声

### 元素攻击 (Element Attack)
- Fire: 橙红色火焰冲击
- Water: 蓝色水流旋涡
- Wind: 青白色风刃
- Earth: 棕灰色岩石
- Lightning: 金黄色电弧

### 虚空攻击 (Void Attack)
- 颜色: 紫色/暗红渐变
- 特效: 扭曲的空间裂隙
- 粒子: 暗影粒子下沉

## 图集打包规范

### 打包格式
```
单敌人图集: enemy_[id]_atlas.png
主图集: enemies_atlas.png (包含所有敌人)

布局: 水平排列，每帧64px
图集尺寸: 64px × (总帧数 × 64px)
```

### 命名示例
```
enemy_cloud_wisp_atlas.png    (64×256, 4帧idle)
enemy_void_lord_atlas.png     (128×2048, 16帧战斗循环)
enemies_basic_atlas.png       (基础敌人合集)
enemies_elite_atlas.png       (精英敌人合集)
enemies_boss_atlas.png        (BOSS敌人合集)
```
