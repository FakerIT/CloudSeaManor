# 人物随时间发展系统

> 本系统定义云海山庄中20位可婚角色随游戏时间推进而产生的行为变化、对话演进、关系深化与环境交互。
>
> 目标：让角色"活"在山庄中，有自己的生活节奏、情感变化和成长轨迹。

---

## 一、总览

### 1.1 设计目标

- **真实性**：角色有自己的作息、职业劳动和社交活动，不只是等待玩家。
- **成长感**：角色会记住玩家做过的事，关系越深，互动越丰富。
- **差异性**：每位角色的时间行为完全不同，符合其职业与性格。
- **可探索性**：玩家可以通过观察角色的时间行为，发现新的事件和好感度入口。

### 1.2 时间维度

| 维度 | 刻度 | 说明 |
|------|------|------|
| 日内 | 4个时段 | 早晨 6:00-11:00 / 午后 11:00-17:00 / 傍晚 17:00-20:00 / 夜间 20:00-24:00 |
| 周内 | 7天循环 | 部分角色有周常行为（如每周某日集市出摊） |
| 季内 | 4季循环 | 角色劳动内容、衣着、对话随季节显著变化 |
| 年内 | 节气/节日 | 24节气+传统节日触发专属事件与对话 |
| 长期 | 好感阶段 | 1-3心 / 4-6心 / 7-9心 / 10心 四个阶段的互动深度递进 |

### 1.3 角色状态系统

每个角色拥有以下时间相关状态：

```
角色状态 = {
    当前时段位置,    // 角色当前在地图上的哪个位置
    当前活动状态,    // 工作中 / 休息中 / 社交中 / 特殊事件中
    近日访问记录,    // 记录玩家最近3次访问该角色的时间和对话主题
    季节熟悉度,      // 该角色对当前季节的熟悉程度，影响季节对话
    长期记忆片段,    // 记录玩家曾做过的关键事件，影响后续对话提及
    情感阶段标记,    // 当前好感阶段，决定对话深度和可触发事件
}
```

### 2.2 时段触发规则

1. **正常时段访问**：玩家前往角色所在位置，触发对应时段对话。
2. **错时访问**：若玩家在非该角色常规时段访问，触发特殊对话（如"你怎么这么早？"）。
3. **跨时段追踪**：若玩家在同一角色的连续3个时段都访问，解锁"你今天来了好多次"系列台词。
4. **时段内多次访问**：同时段内第2次访问，触发简短寒暄台词；第3次触发"你是不是有什么事？"关心台词。

### 2.3 时段对话进化

以清荷为例，展示好感阶段对时段对话的影响：

**好感 1-3 心（初识期）**
```
早晨（清荷）："早……早上好。今天荷叶上露珠很重，你来得正好。"
午后（清荷）："下午好。你又来荷塘了？"
傍晚（清荷）："傍晚好。今天晚霞很美呢。"
夜间（清荷）："晚上好……这么晚了，你一个人来荷塘，不害怕吗？"
```

**好感 4-6 心（熟悉期）**
```
早晨（清荷）："早安。我正想去找你呢，没想到你先来了。"
午后（清荷）："你来啦？我今天做了新的莲子糕，来尝尝？"
傍晚（清荷）："今天的晚霞和那天一样美……你要不要一起看？"
夜间（清荷）："这么晚了你还来……我就知道你会来的。"
```

**好感 7-9 心（亲密期）**
```
早晨（清荷）："你今天来得比往常早呢。（笑）我给你留了最新鲜的莲子。"
午后（清荷）："我就知道你会来。茶已经泡好了，坐吧。"
傍晚（清荷）："今天的晚霞很美，不过……有你在，更美。"
夜间（清荷）："等你很久了。我煮了莲子粥，进来暖暖身子吧。"
```

**好感 10 心（恋人期）**
```
早晨（清荷）："（看到你，脸上泛起红晕）你……你今天来得最早了。"
午后（清荷）："你来的时候，整个荷塘都亮了起来呢。"
傍晚（清荷）："每天这个时候，我都在想，你会不会来。你每次都来了。"
夜间（清荷）："（在屋内等待，听到脚步声开门）我就知道是你。"
```

## 三、季节行为系统

### 3.1 四季场景变化

| 季节 | 场景色调 | 角色衣着 | 劳动内容 | 特色对话主题 |
|------|---------|---------|---------|------------|
| 春 | 嫩绿+桃花粉 | 轻薄春装 | 春播、新茶 | "春耕"、"清明茶会"、"花朝节" |
| 夏 | 翠绿+荷花粉 | 轻薄夏装 | 采莲、果熟 | "萤火虫"、"夏至"、"七夕" |
| 秋 | 金黄+枫红 | 温和秋装 | 丰收、酿酒 | "重阳"、"秋收"、"桂花" |
| 冬 | 灰白+雪白 | 厚实冬装 | 储藏、休息 | "雪景"、"围炉"、"春节" |

### 3.2 角色季节专属行为

#### 春季节专属
- **清荷**：春季荷塘刚开始发芽，会在塘边补种新莲藕。与她对话会聊到"去年冬天荷塘冻伤了，今年要好好照料"。
- **桃夭**：春季最忙，花朝节前后要布置整个花坡。可触发帮忙种花的合作事件。
- **林风**：春季百花盛开，蜜蜂最忙，蜂蜜产量最高。会说"今年花期来得早"。
- **松年**：春季老松树发新芽，会收集松针做茶。

#### 夏季节专属
- **清荷**：夏季荷花开得最盛，会泛舟采莲。夜晚萤火虫活动。
- **杏儿**：夏季桃子、杏子成熟，产果量最大的季节。
- **云舒**：夏季云海最壮观，是写生的旺季。
- **石生**：夏季石头热胀冷缩，不宜做大型雕刻，多做精细活。

#### 秋季节专属
- **秋棠**：秋季酿酒高峰期，桂花酒、菊花酒上市。
- **林风**：秋季蜂蜜最浓郁，采集百花蜜。
- **松年**：秋季落叶最美，与玩家一起打扫落叶，下棋。
- **云舟**：秋季鱼肥，河面平静，撑船最舒服。

#### 冬季节专属
- **清荷**：冬季荷塘休眠，但莲藕最甜，可挖藕。
- **石生**：冬季石头最硬，适合做奠基之石。
- **雾川**：冬季云海最凝重，守护任务最重。
- **松年**：冬季雪后松树最美，棋亭围炉品茶。

### 3.3 季节对话模板

以清荷为例，展示四季同一时段的对话差异：

**傍晚时段**
- 春："春天的荷塘刚发芽，嫩绿嫩绿的。你看那边，有小蜻蜓了。"
- 夏："夏天的晚霞最漂亮，你看那边，红得像织女的绸缎。"
- 秋："秋天的荷叶开始枯了，但莲藕长成了。你喜欢吃莲藕吗？"
- 冬："冬天荷塘安静了，但雪后的荷塘别有一番滋味呢。"

## 四、天气行为系统

### 4.1 四种天气类型

| 天气 | 云海状态 | 灵气浓度 | 角色反应 |
|------|---------|---------|---------|
| 晴 | 清澈透亮 | 低 | 大多数角色开心，劳动效率高 |
| 薄雾 | 轻纱笼罩 | 中 | 气氛朦胧，部分角色有诗意对话 |
| 浓云海 | 翻涌翻腾 | 高 | 灵气相关角色（晚星、玄清等）有专属反应 |
| 大潮 | 潮涌云天 | 极高 | 全体角色有特殊反应，解锁稀有事件 |

### 4.2 天气特殊事件

#### 大潮日专属
- **清荷**：大潮夜的萤火虫会格外明亮，可在荷塘触发"萤火告白"事件。
- **桃夭**：大潮天花朵吸收灵气，花酿品质提升，可触发"灵花秘酿"事件。
- **晚星**：大潮夜灵气最强，解锁"灵界观测"特殊功能。
- **玄清**：大潮夜灵界波动最大，需加强巡逻，可触发"共守灵界"事件。
- **老槐**：大潮时能量最强，可触发"百年对话"——讲述山庄更古老的历史。
- **阿巳**：大潮夜山中灵气最浓，可带玩家去"灵气洞穴"探险。

### 4.3 天气对话示例

**石生的天气对话随好感阶段变化：**

好感 1-3 心（简短、就事论事）
```
晴："今天天气好，适合开石。"
薄雾："有雾，不影响干活。"
浓云海："灵气重了些。"
大潮："今天不适合干精细活。"
```

好感 4-6 心（稍微主动，关心玩家）
```
晴："晴天干活痛快。你今天也精神得很。"
薄雾："这种天气，我干活最舒服。你呢？"
浓云海："云海浓了……你进山要小心。"
大潮："大潮来了，你不要乱跑，我去看看矿洞。"
```

好感 7-10 心（关心、邀请）
```
晴："难得的好天气……想不想和我一起去山里看看？"
薄雾："这种雾……我小时候最喜欢。你小时候喜欢做什么？"
浓云海："云海这么浓，石头会有灵性。你信不信？"
大潮："大潮的日子，整个山都在呼吸……陪我看看？"
```

## 五、长期记忆与关系演进

### 5.1 记忆片段系统

角色会记住玩家做过的重要事件，并在后续对话中自然提及：

| 记忆类型 | 示例 | 后续提及方式 |
|---------|------|------------|
| 帮助过她 | 帮她修好了荷塘的堤 | "上次你帮我修堤……我一直记着呢。" |
| 送过礼物 | 送了她一束野花 | "你上次送的花，我还留着呢，做成干花了。" |
| 说过的话 | 玩家承诺某天再来 | "你上次说会来，我等了你三天……" |
| 共同经历 | 一起看过萤火虫 | "又到萤火虫的季节了……今年还去看吗？" |
| 关心过她 | 在她生病时送过药 | "那次你送来的药……我一直留着。" |

### 5.2 关系阶段演进

#### 第一阶段：初识（好感 1-3 心）
- 对话礼貌、客气，有固定的开场白
- 每次访问都需要重新破冰
- 不会主动邀请玩家去自己的私人空间
- 同性主角：友好但保持距离

#### 第二阶段：熟悉（好感 4-6 心）
- 开始有主动的关心和问候
- 愿意分享自己的工作和生活
- 会主动邀请玩家帮忙或去私人空间
- 同性主角：开始建立闺蜜/兄弟情谊

#### 第三阶段：亲密（好感 7-9 心）
- 有明显的情感依赖和期待
- 会在玩家没来时主动提及想念
- 愿意分享更深层的秘密和心事
- 同性主角：挚友关系，互相关心

#### 第四阶段：确立（好感 10 心）
- 明确的情感表达（恋爱/知己）
- 日常对话充满默契
- 愿意为玩家做特别的事
- 触发求婚/深度知己事件

### 5.3 关系演进对话示例（云舒）

**阶段一（初识）**
```
云舒："你好，又来写生了吗？"
```

**阶段二（熟悉）**
```
云舒："你来啦。我今天画了一幅云海，很想听听你的意见。"
```

**阶段三（亲密）**
```
云舒："你今天来得有点晚……我一直在等你。"
（稍微停顿）
"我刚才在画你。你来了正好，帮我看看像不像。"
```

**阶段四（恋人）**
```
云舒："（抬头看到你，放下画笔，眼睛亮起来）"
"我就知道今天你会来。"
"你来的时候，我正好画到这里……你看。"
（画面上是玩家走进来的背影，背景是云海）
"每一幅画里，都有你。"
```

| 表亲 | 杏儿 ↔ 林风 | 表兄妹关系（杏儿母亲=林风姑姑） |
| 画友 | 云舒 ↔ 锦时 | 文艺交流，云舒的画配锦时的诗 |

### 7.2 社交事件触发

- **偶遇事件**：玩家跟踪某角色时，可能遇到其他角色，触发随机对话。
- **合作事件**：特定节日触发两人合作（如七夕晚星+阿巳观星会）。
- **矛盾事件**：部分角色间有小矛盾，可触发帮助调解的好感事件。
- **三人对话**：玩家在场时，角色间会互相打招呼，形成自然的社交氛围。

### 7.3 社交对话示例

**杏儿遇到林风**
```
杏儿："表哥！你今天蜂蜜收了没？"
林风："收了收了！你那杏子今年甜不甜？"
杏儿："可甜了！回头我给你送一筐！"
林风："成交！我给你留两罐最好的蜂蜜！"
```

**玩家在场时，阿茶遇到秋棠**
```
阿茶："秋棠姐，今天的酒酿得怎么样了？"
秋棠："阿茶妹妹！你来得正好，刚酿了一坛桂花茶酒，你来尝尝？"
阿茶："好啊，正好我带了今年的新茶，我们一起品品？"
```

## 八、特殊时间事件

### 8.1 连续访问惩罚

- 同一角色，同一天连续访问超过5次：从第5次起，每次对话好感+1（正常为+10），触发"你今天怎么来了这么多次？"台词。

### 8.2 长时间未访问

| 未访问天数 | 角色反应 | 好感度变化 |
|-----------|---------|-----------|
| 3天 | "你最近很忙吗？" | 无变化 |
| 7天 | "好几天没见到你了……" | 无变化 |
| 14天 | "你是不是忘了来看我？" | 好感-5（最低到1心） |
| 30天 | "……你终于来了。"（冷淡） | 好感-10，触发"疏远期"对话 |
| 60天 | "你来做什么？"（陌生） | 好感-20，需要完成"重新认识"事件恢复 |

### 8.3 特定时间访问奖励

- **清晨最早**：玩家是当天第一个与该角色说话的人，解锁"清晨专属台词"，好感额外+2。
- **深夜最后**：玩家在夜间最后时段访问，角色会有更私密的心事对话。
- **雨中访问**：雨天主动去找角色，展现关心，好感额外+3。
- **大潮夜访问**：大潮夜去找晚星/玄清/老槐等角色，解锁稀有剧情。
