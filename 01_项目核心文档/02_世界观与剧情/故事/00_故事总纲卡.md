## 作品概述

### 作品定位

**《云海山庄》**是一款治愈系生活模拟与轻度奇幻冒险相结合的独立游戏。玩家扮演一位继承祖父遗产的年轻人，回到荒废已久的云海山庄，在经营农场、发展人际关系、探索神秘世界的过程中，逐渐揭开家族的秘密，并肩负起守护三界平衡的使命。

### 核心卖点

| 卖点类型 | 具体内容 | 目标受众 |
|---------|---------|---------|
| 情感治愈 | 温馨的人际关系，缓慢的生活节奏 | 压力大的都市人群 |
| 农场经营 | 灵茶种植、灵兽培育、建筑升级 | 经营模拟爱好者 |
| 奇幻探索 | 界桥遗迹、上界冒险、灵界冒险 | 冒险游戏爱好者 |
| 多结局叙事 | 10+结局分支，多角色恋爱线 | 角色扮演爱好者 |
| 重玩价值 | 多周目继承，新内容解锁 | 核心向玩家 |

### 游戏定位

| 维度 | 定位 |
|-----|------|
| 平台 | PC（Steam优先）/ Nintendo Switch |
| 类型 | 生活模拟 + 轻度冒险 |
| 画风 | 东方水墨风格 + 治愈系 |
| 目标时长 | 35-50小时（一周目） |
| 目标人群 | 18-35岁，喜欢治愈系/农场游戏 |

## 世界观设定

### 世界观架构

云海山庄的故事发生在一个人界、灵界、上界三界共存的世界中。三界通过"界桥"相连，灵气在各界之间流转，维持着世界的平衡。

### 云海山庄的起源

**建立时间**: 三千年前

**建立者**: 云隐真人（姓氏不详，以"云"为号）

**建立背景**:
三千年前，三界处于混沌状态，灵气混乱流淌，人间灾祸不断。一位名叫云隐真人的修行者为拯救苍生，以毕生修为开辟了"界桥"，将三界分离并建立灵气循环。云隐真人在凡间建立山庄，作为界桥的守护枢纽。

**历史沿革**:

| 时期 | 时间跨度 | 守护者 | 主要事件 |
|-----|---------|-------|---------|
| 初创期 | 0-500年 | 云隐真人 | 建立山庄，开创守护体系 |
| 繁荣期 | 500-1500年 | 历代守护者 | 三界平衡，山庄繁荣 |
| 动荡期 | 1500-2500年 | 历代守护者 | 界桥多次失衡，经历危机 |
| 近现代 | 2500-3000年 | 云海真人 | 封印大潮，牺牲自己 |
| 当代 | 3000年+ | 玩家（继承人） | 故事开始 |

### 界桥的本质

**界桥**是连接三界的通道，也是灵气循环的枢纽。它的状态直接影响三界的稳定。

**界桥的三个状态**:

| 状态 | 描述 | 对三界的影响 |
|-----|------|-------------|
| 平衡态 | 界桥稳定运转，灵气均衡流动 | 三界和平，繁荣发展 |
| 开启态 | 界桥大开，灵气快速流动 | 凡间灵气充沛但不稳定 |
| 关闭态 | 界桥封闭，灵气滞留 | 凡间灵气枯竭但稳定 |

**界桥的核心**:
界桥的核心是一个巨大的灵气旋涡，位于云海山庄山巅的太初云海遗迹中。历代守护者通过控制核心来调节三界的灵气平衡。

**界桥的代价**:
维持界桥需要消耗守护者的生命灵气。历代守护者都会在一定周期后"羽化"，将生命融入界桥以维持其运转。

### 灵气的本质

**灵气**是维系三界运转的根本能量，存在于一切有形与无形之物中。

**灵气的特性**:
- 无色无形，但可以被特定体质的人感知
- 会自然聚集在灵气充沛的地点（灵穴）
- 可以被特定植物吸收（灵植）
- 可以被特定生物吸收（灵兽）
- 可以被人类修炼提升（修行者）

**灵气与人的关系**:
- 普通人不感知灵气，但受其益处
- 修行者可以感知并运用灵气
- 守护者血脉与灵气有特殊共鸣
- 灵气过高或过低都会影响人类健康

### 三界详解

#### 凡间（云海）

**定义**: 三界中灵气浓度最低的一界，但最适合人类生活。

**特点**:
- 四季分明，气候温和
- 与其他两界以界桥相连
- 灵气稀薄但稳定
- 人类为主要种族

**云海山庄所在**:
云海山庄位于凡间的灵气核心位置，是界桥在凡间的显化。山庄周围的云海是灵气聚集形成的独特景观。

#### 上界（仙界）

**定义**: 灵气浓度最高的一界，修行者的圣地。

**特点**:
- 云雾缭绕，四季如春
- 灵气充沛但不均衡
- 存在多个仙门流派
- 以茶道、符术、剑道为主要修行方式

**主要势力**:
- 茶仙派：以茶道通神，传承最古老
- 符箓宗：以符术闻名，擅长结界
- 剑修盟：以剑道立身，战斗力最强
- 天机阁：以占星术预测未来

**下凡者**:
由于某些原因，部分上界之人会落入凡间。茶九就是其中之一。

#### 下界（深渊）

**定义**: 灵气混乱的一界，贪婪者的聚集地。

**特点**:
- 暗无天日，资源匮乏
- 灵气混乱但充沛
- 存在试图攫取灵气者
- 与上界对立

**主要势力**:
- 贪婪者联盟：试图关闭界桥，独占灵气
- 深渊恶魔：试图打破界桥，释放混沌
- 守护氏族：如雾川所在的山林氏族，世代守护界桥

## 主线故事概要

### 故事起点

玩家是一位在城市生活的年轻人，因祖父去世而回到阔别多年的云海山庄。这里曾是祖父经营的山庄，如今已荒废多年。玩家本打算处理完遗产事宜后返回城市，却在这里发现了改变命运的秘密。

### 故事主题

**核心主题**: 从逃离到回归，从继承到守护，从孤独到羁绊

**叙事主线**:
```
第一阶段：回归与适应
→ 玩家回到山庄，面对荒废的土地和村民的怀疑
→ 逐渐恢复山庄经营，学习灵茶种植和灵兽培育
→ 与旧友重逢，建立新的羁绊

第二阶段：使命与传承
→ 发现祖父的遗物《云海录》
→ 了解界桥和守护者的历史
→ 遭遇第一次大潮，初步感知危机

第三阶段：真相与抉择
→ 深入探索太初遗迹
→ 了解三界势力的争夺
→ 面临第一次重大抉择

第四阶段：决战与结局
→ 整合资源和人脉
→ 面对最终决战
→ 做出最终选择，达成结局
```

### 核心冲突

**主线冲突**: 界桥的平衡与命运

| 冲突方 | 立场 | 诉求 |
|-------|------|------|
| 上界激进派 | 彻底打开界桥 | 获取更多灵气，传播仙道 |
| 下界贪婪者 | 彻底关闭界桥 | 独占凡间资源 |
| 混沌守望者 | 毁灭界桥 | 终结一切痛苦 |
| 玩家与村民 | 维持平衡 | 保护家园，守护三界 |

**个人冲突**: 归属与责任的抉择

| 冲突方 | 立场 |
|-------|------|
| 城市生活 | 玩家原本的生活和未来 |
| 云海山庄 | 祖父的遗产和村民的期望 |
| 内心的羁绊 | 与角色们的深厚感情 |

### 核心命题

**三大命题**:

1. **人心与灵气的共存**
   - 灵气是维持世界的能量，但人心的贪婪可能破坏平衡
   - 人类能否在追求发展的同时保护灵气？

2. **传承与创新的平衡**
   - 传统需要守护，但时代在变化
   - 守护者是否应该墨守成规，还是与时俱进？

3. **个人与群体的抉择**
   - 爱情、友情、使命之间如何取舍？
   - 为了更大的目标，个人可以牺牲多少？

## 叙事支柱

### 叙事支柱一：回归与治愈

**核心内容**:
游戏以"回归"为主题，玩家从繁华的都市回到宁静的山庄，寻找心灵的慰藉。这是一个关于"家"的故事——无论走多远，最终都会回到最初的地方。

**叙事特点**:
- 缓慢的生活节奏，强调日常的美好
- 与角色建立深厚的情感联系
- 通过劳动和付出获得成就感和归属感
- 治愈的心灵创伤（玩家的和角色的）

**典型场景**:
- 清晨在茶园采茶
- 傍晚在观云台观云海
- 夜晚与角色在亭中饮茶谈心
- 节日庆典与村民共度

### 叙事支柱二：羁绊与成长

**核心内容**:
游戏强调"羁绊"的力量——人与人之间的联系是人世间最珍贵的东西。通过与角色的互动，玩家不仅帮助角色成长，自己也在这个过程中成长。

**叙事特点**:
- 每个角色都有完整的成长弧线
- 玩家的选择影响角色的命运
- 角色之间的互动和关系变化
- 从陌生到亲密的关系发展

**典型场景**:
- 阿茶从等待到主动表白
- 雾川从冷漠到敞开心扉
- 茶九从不羁到甘愿平凡
- 小满从迷茫到坚定梦想

### 叙事支柱三：守护与责任

**核心内容**:
游戏探讨"守护"的含义——守护不是占有，而是为了让美好事物延续。这是一种传承，也是一种牺牲。

**叙事特点**:
- 历代守护者的故事
- 玩家逐渐理解祖父的选择
- 最终自己成为守护者
- 传承的意义：不是结束，而是延续

**典型场景**:
- 松年讲述历代守护者的故事
- 玩家发现祖父的手记
- 雾川揭示山林氏族的使命
- 最终决战中的牺牲与传承

### 叙事支柱四：选择与代价

**核心内容**:
游戏强调"选择"的重要性——每一个选择都有代价，没有完美的答案。玩家的选择塑造了故事的走向，也塑造了角色们的命运。

**叙事特点**:
- 多重选择的分支
- 选择的后果在很久之后才显现
- 没有绝对正确或错误的选择
- 每个选择都带来不同的故事

**典型场景**:
- 第5章的三方势力选择
- 第6章的最终抉择
- 各种角色事件的分支选择
- 日常对话中的微妙选择

## 游戏系统与叙事绑定

### 关系系统

**系统说明**:
关系系统是游戏的核心系统之一，通过与角色互动提升好感度，解锁角色剧情和专属结局。

**好感度等级**:

| 等级 | 心级 | 可解锁内容 |
|-----|------|----------|
| 陌生 | 0-1心 | 基础对话 |
| 熟悉 | 2-3心 | 日常委托 |
| 友好 | 4-5心 | 角色背景 |
| 亲密 | 6-7心 | 深层剧情 |
| 挚友 | 8-9心 | 10心前置 |
| 誓约 | 10心 | 求婚/结婚 |

**好感获取方式**:
- 对话选择
- 赠送礼物
- 完成任务
- 共同战斗
- 剧情事件

**好感影响**:
- 角色对话和态度
- 角色支援效果
- 终局可进入的路线
- 结局的达成条件

### 经营系统

**系统说明**:
经营系统是游戏的主要玩法，通过种植灵茶、培育灵兽、升级建筑来发展山庄。

**核心资源**:

| 资源 | 用途 | 获取方式 |
|-----|------|---------|
| 灵气 | 核心指标，影响一切 | 种植/灵兽/维护 |
| 金币 | 经济资源 | 出售/任务/委托 |
| 灵石 | 高级货币 | 探索/战斗 |

**建筑系统**:

| 建筑 | 功能 | 影响 |
|-----|------|------|
| 主屋 | 居住/存档 | 山庄等级 |
| 茶园 | 种植灵茶 | 灵气收入 |
| 灵兽园 | 培育灵兽 | 灵气收入/战斗 |
| 观云台 | 观云/占星 | 剧情解锁 |
| 仓库 | 存储物品 | 容量上限 |

**建筑升级**:
- 每栋建筑有5个等级
- 升级需要材料和金币
- 升级后功能增强
- 部分升级是剧情前置

### 探索系统

**系统说明**:
探索系统让玩家进入灵界和上界，发现新资源和剧情。

**探索区域**:

| 区域 | 位置 | 特点 |
|-----|------|------|
| 灵界边缘 | 山庄周边 | 入门级，适合新手 |
| 太初遗迹 | 山巅 | 主线剧情，需要钥匙 |
| 上界通道 | 界桥处 | 茶九线，需要特定条件 |
| 下界深渊 | 山脚 | 雾川线，需要封印解除 |

**探索要素**:
- 界桥碎片：终局关键道具
- 隐藏灵兽：图鉴收集
- 古代遗物：剧情道具
- 灵气节点：恢复灵气

### 战斗系统

**系统说明**:
战斗系统用于对抗灵界怪物和保护山庄。

**战斗类型**:

| 类型 | 说明 | 难度 |
|-----|------|------|
| 野外遭遇 | 随机触发 | 低 |
| 任务战斗 | 主线/支线 | 中 |
| Boss战 | 大潮/Boss | 高 |

**战斗要素**:
- 玩家角色控制
- 灵兽助战
- 配偶/角色支援
- 环境互动

## 玩家能动性

### 选择的影响

**日常选择**:
- 对话选择影响角色好感（细微影响）
- 任务选择影响资源获取
- 时间分配影响发展速度

**关键选择**:
- 角色事件选择影响角色发展
- 势力立场选择影响剧情走向
- 最终抉择决定结局类型

### 自由度高

**自由度体现**:
- 建设顺序自由
- 角色攻略顺序自由
- 探索顺序自由
- 每日活动自由

**自由度限制**:
- 部分内容有前置要求
- 季节限定事件
- 体力/时间限制

### 玩家塑造角色

**角色塑造方式**:
- 选择与谁深入交往
- 选择支持谁的立场
- 选择帮助谁度过危机
- 选择与谁共度余生

## 叙事风格

### 整体基调

**治愈温暖**: 游戏整体基调是温暖治愈的，即使有冲突和危机，最终也会以正面的方式解决。

**日常美好**: 强调日常生活中的小确幸，而非宏大的英雄叙事。

**情感细腻**: 角色之间的情感描写细腻真挚，让玩家产生代入感。

### 语言风格

**对话**: 符合角色性格，简洁有力，富有情感。

**旁白**: 诗意化的描述，营造意境。

**系统提示**: 轻松幽默，提供指引。

### 视觉叙事

**场景设计**: 东方水墨风格，云雾缭绕，诗意盎然。

**角色立绘**: 细腻的表情变化，体现情感波动。

**CG场景**: 关键剧情的视觉高潮，强调仪式感和情感冲击。
