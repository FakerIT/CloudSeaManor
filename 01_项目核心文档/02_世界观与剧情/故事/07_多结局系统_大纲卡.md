# 多结局系统｜大纲卡

> **系统类型**: 核心叙事分支系统
> **相关章节**: 第6章（终章）
> **结局总数**: 10个主要结局 + 3个变体结局 + 1个隐藏结局
> **设计目标**: 体现玩家选择的长期影响，提供高重玩价值

---

## 结局系统概述

### 设计理念

云海山庄的多结局系统基于"玩家行为累积"原则——玩家在整个游戏过程中的每一个选择都将影响最终的结局走向，而非仅在终章做出最终二选一。这种设计确保了：
- **叙事一致性**: 结局是整个旅程的自然终点，而非凭空出现
- **情感深度**: 玩家在前期建立的羁绊将在结局中得到回报
- **重玩价值**: 不同的选择组合将导向不同的结局，鼓励多周目体验

### 结局分类

| 类别 | 数量 | 描述 |
|-----|------|------|
| 真结局 | 1 | 最高难度，需满足全部高标准条件 |
| 角色结局 | 5 | 基于特定角色10心+特定路线 |
| 特殊结局 | 2 | 非恋爱向的独特结局 |
| Bad End | 2 | 失败/负面结局 |
| 隐藏结局 | 1 | 需要满足额外隐藏条件 |
| 变体结局 | 3 | 基于同一路线但不同表现 |

### 结局与系统绑定

| 游戏系统 | 对结局的影响 |
|---------|-------------|
| 灵气值 | 直接决定可达成结局类型 |
| 角色好感 | 决定可进入的角色线结局 |
| 界桥碎片 | 决定可选择的终局路线 |
| 建筑状态 | 影响真结局达成条件 |
| 战斗评价 | 影响最终CG与评价 |

"（若已婚）我为你骄傲。"
