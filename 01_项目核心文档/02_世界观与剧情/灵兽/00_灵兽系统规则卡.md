# 灵兽系统规则卡

- 可收服总数：30种（原有12种 + 新增云/风/光/土/露/潮/暗七系各3种）
- 可进化：全部支持三阶段进化
- 可繁殖：成年灵兽 + 灵界莲×1 + 云海浓度达标
- 同时携带上限：6只（灵兽园高阶后可突破）
- 功能定位：农活、战斗、社交、事件触发

## 核心机制
- 进化条件：特定材料 + 云海浓度 + 灵兽好感
- 好感提升：喂食、共同工作、战斗胜利
- 10心收益：解锁灵兽专属心事件
- 可关闭战斗：支持纯生活向游玩

## 元素克制体系
| 元素 | 克制 | 被克制 |
|------|------|--------|
| 风 | 云 | 露 |
| 云 | 露 | 光 |
| 露 | 光 | 潮 |
| 光 | 潮 | 风 |
| 潮 | 风 | 云 |
| 土 | 全元素 | 无 |
| 暗 | 无 | 光 |
| 中性 | 无 | 无 |

## 隐藏彩蛋
- 集齐30种并完成太初雏龙终进化，触发"灵兽大典"全村庆典。
- 集齐同系3种灵兽，触发"元素共鸣"特殊事件。
- 暗系灵兽与光系灵兽同时携带，触发"光暗交织"隐藏剧情。
