## 元素克制体系

```
       风 → 云 → 露 → 光 → 潮 → 风
             ↓       ↓
           土(中立)  暗(无克)
```

| 元素 | 克制 | 被克制 |
|------|------|--------|
| 风 | 云、露 | 光、潮 |
| 云 | 露 | 风 |
| 露 | 光 | 潮、风 |
| 光 | 潮 | 风 |
| 潮 | 风 | 云 |
| 土 | 全元素 | 无 |
| 暗 | 无 | 光 |
| 中性 | 无 | 无 |
