## 节日详细概述

### 春季节日

#### 01_山庄苏醒祭

**关键词：** 开局首个节日 / 主屋修缮 / 全村协作 / 邻里互助

山庄苏醒祭是春季第一个节日，象征着寒冬过后山庄的重新苏醒。玩家继承山庄后的第一个正式节日，全村人聚在一起清理积雪、修补建筑、装饰环境。核心是选择帮忙任务与村民互动，傍晚庆典烟火绽放照亮夜空。

**特色：** 温馨热闹的开局氛围，所有角色好感+20，是建立村民关系的重要时机。

#### 02_灵茶初芽节

**关键词：** 灵茶系统 / 采茶竞赛 / 茶九CG / 阿茶感情线

灵茶初芽节专为庆祝首批灵茶新芽萌发而设，只在浓云海天气下触发。核心是采茶竞赛小游戏（需在云海散去前完成）和制茶体验。下午茶会可送茶给心仪角色，傍晚与阿茶独处观云海。

**特色：** 茶香袅袅的清新氛围，阿茶和茶九感情线的关键节点，可获得茶九专属CG。

#### 03_春耕祈福

**关键词：** 灵兽协同耕地 / 祈福仪式 / 随机灵兽蛋 / 小满感情线

春耕祈福是山庄农业年度开端。玩家与灵兽协同耕地，灵兽可能随机发现隐藏物品（灵草种子/灵石）。正午祈福仪式后全农田产量+20%持续春季。表现最好的灵兽获得随机灵兽蛋。

**特色：** 生机勃勃的春耕氛围，全游戏唯一稳定获取灵兽蛋的节日。

### 夏季节日

#### 04_云海灯会

**关键词：** 浪漫灯会 / 星星灯笼 / 晚星感情线 / 河灯许愿

云海灯会是夏季最浪漫节日，当夜幕降临整个山庄被灯笼点亮。玩家制作灯笼（可选星星灯笼）并在观星台与晚星一同挂灯。下午可邀请角色共赏灯会，晚星6-8心事件在此解锁。

**特色：** 灯火与云海相映的人间仙境氛围，晚星感情线的核心节日。

#### 05_星河夜泳

**关键词：** 温泉夜泳 / 秋棠感情线 / 星空倒映 / 配偶二人世界

星河夜泳从傍晚开始，玩家与秋棠、清荷、烟萝等在温泉中享受星空。深夜可与被邀请者深度对话，未婚玩家可推进感情线，已婚玩家可触发配偶专属场景。

**特色：** 浪漫温馨的星空温泉氛围，夏季节日中唯一的配偶专属节日。

#### 06_夏日灵兽赛

**关键词：** 灵兽才艺展示 / 竞速比赛 / 小满感情线 / 灵兽换装

夏日灵兽赛是灵兽展示才华的舞台，包含才艺展示、竞速比赛、换装表演和颁奖典礼。玩家携灵兽参赛，小满会对优秀灵兽表现出崇拜，是小满感情线的关键节日。

**特色：** 活力四射的童趣氛围，小满高阶剧情的解锁点。

### 秋季节日

#### 07_丰收宴

**关键词：** 烹饪比赛 / 全村聚餐 / 社交高潮 / 繁荣度提升

丰收宴是秋季最重要的节日，庆祝一年劳作收获。核心是一场烹饪比赛——玩家选择菜品并与角色协作完成。下午品鉴后老槐会说「因为这道菜里，有大家的心意」，全角色好感+30。

**特色：** 热闹温馨的丰收氛围，全年中最重要的社交节日。

#### 08_灵兽大比

**关键词：** 灵兽竞技 / 界影狼幼崽 / 雾川感情线 / 传说级灵兽

灵兽大比比夏日灵兽赛更侧重实力展示。比赛进行中会触发隐藏事件——受伤的界影狼幼崽出现。玩家可选择救它（获得传说级灵兽）或放它离开。雾川对玩家的选择给予肯定。

**特色：** 史诗感十足的守护主题，界影狼是全游戏最重要的隐藏事件之一。

#### 09_秋祭忆故

**关键词：** 祭拜先人 / 主线剧情揭示 / 玄清感情线 / 主线分歧

秋祭忆故是最肃穆的节日，祭拜已故的外公。玄清揭示《云海录》内容和世界观设定——山庄建立在人界与灵界交汇处。下午观云台对话揭示玩家作为守护者的使命，是玄清感情线的核心节日。

**特色：** 庄重肃穆的缅怀传承氛围，主线第4章的关键节点。

---

### 冬季节日

#### 10_雪雾封山

**关键词：** 大雪封山 / 居家温馨 / 素锦/烟萝CG / 配偶二人世界

雪雾封山当大雪降临、浓云海笼罩时触发。上午玩家选择帮忙准备封山（触发素锦/烟萝/小满/老槐好感事件），下午可与素锦做针线活（获专属CG）或与烟萝煮药。已婚玩家可触发配偶二人世界。

**特色：** 寒冷中的温暖居家氛围，素锦和烟萝CG的获取节日。

#### 11_守岁祈福

**关键词：** 除夕守岁 / 新年倒计时 / 全年最浪漫时刻 / 多结局前置

守岁祈福是一年最重要的节日——除夕夜全村守岁迎接新年。午夜所有人一起倒数，烟火绽放，每个角色说出新年第一句话。全年情感浓度最高的节日，也是多结局系统的前置条件。

**特色：** 温馨感人的辞旧迎新氛围，全游戏最浪漫的时刻。

---

### 跨季节节日

#### 12_云海大潮

**关键词：** 百年一遇 / 界桥修复 / 主线第5章 / 太初雏龙蛋

云海大潮是第2-3年、灵气>=600时触发的限次事件（一生一次）。活动分三阶段：黎明准备、穿越云海修复界桥、危机解除。可获得太初雏龙蛋（传说级灵兽），主线推进到第5章。

**特色：** 史诗壮观的紧张刺激，是整个游戏最重要的里程碑之一。

#### 13_游客开放日

**关键词：** 客栈经营 / 小游戏评分 / 传说种子 / 每季度可触发

游客开放日在客栈>=2级时每季度触发一次。玩家参与接待、餐饮、讲解或销售工作，满足游客需求小游戏根据评分发放不同奖励（完美3000金币+传说种子）。

**特色：** 热闹经营的田园风情，后期稳定的经济来源。

#### 14_界桥观测夜

**关键词：** 主线分歧 / 三结局路线 / 三选一观测 / 关键抉择

界桥观测夜在主线第4章后触发（限次）。玩家选择观测人界/灵界/界桥，根据观测结果选择结局走向：修复界桥（真结局）、保持现状（普通结局）或探索更深（隐藏结局）。

**特色：** 神秘深邃的星空哲思，是影响整个游戏结局走向的核心节日。

#### 15_山庄集体婚礼

**关键词：** 多对婚礼 / 百年难遇 / 全角色祝福 / 配偶终极场景

山庄集体婚礼在3位以上角色10心满好感且玩家已结婚时触发（限次）。多对新人同时举办婚礼，所有角色依次送上祝福。未/已婚玩家都能体验到最浪漫的婚礼场景。

**特色：** 浪漫温馨的百年难遇盛典，婚姻系统的终极庆典。

#### 16_太初云海祭

**关键词：** 终极隐藏节日 / 全灵兽聚集 / 三真结局 / 终局

太初云海祭是传说中从未有人见过的终极隐藏节日。触发条件：真结局路线+全灵兽+灵气>=900。所有灵兽聚集，玩家选择结局：永恒守护、灵界探索或新周目。游戏最高潮。

**特色：** 史诗壮观的终极华章，游戏旅程的最高点和终点。
