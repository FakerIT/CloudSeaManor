## 01_山庄苏醒祭

### 节日概述

- **序号**：1
- **季节**：春季
- **触发条件**：春季第1周，灵气>=50
- **持续时间**：全天
- **主要奖励**：主屋升级材料包、全角色好感+20
- **绑定系统**：主屋升级、村民关系线

---

### 节日流程

#### 清晨（6:00-8:00）

**[场景]** 山庄入口处

```
[清晨，山庄入口挂满了彩旗和灯笼，村民们已经开始忙碌]

【通知】"山庄苏醒祭即将开始！请各位村民前往主屋集合！"

阿茶： [正在帮忙挂彩旗]
         "今年的苏醒祭，我准备了好茶给大家喝！"

小满： [扛着锄头从田里跑过来]
         "等等我！我来帮忙搬东西！"

素锦： [拿着针线走过来]
         "我来做一些装饰，大家都辛苦了。"

→ 玩家来到主屋门口
```

#### 上午（8:00-11:00）主屋修缮

**[场景]** 山庄主屋

```
老槐： [站在主屋前，看着破旧的房子]
         "这主屋确实该修缮了，今年的苏醒祭就从这里开始吧。"

老槐： "年轻人，你来当主角。"
         "这是你的山庄，你来带头，大家一起帮忙！"

◆ 玩家选择修缮任务

【选项A】帮忙搬运材料
  → 石生出场
  石生： "我来搬石头，你来指挥放哪儿。"

【选项B】帮忙打扫清理
  → 素锦出场
  素锦： "我来缝补窗帘，你来擦窗户。"

【选项C】帮忙布置装饰
  → 桃夭出场
  桃夭： "来来来！我带了花！布置一下可漂亮了！"

[无论选择哪项，都会触发与对应角色的对话]

石生： "这房子有年头了，我帮你检查一下地基。"
        "放心，有我在，保证住几十年都不会坏。"

素锦： "这窗帘破了，我帮你缝补一下。"
        "你看，缝好了，是不是和新的一样？"

桃夭： "来来来！我种的花插在花瓶里，可漂亮了！"
        "你看，整个屋子都亮起来了！"
```

#### 中午（11:00-13:00）全村聚餐

**[场景]** 山庄广场

```
【通知】"午餐时间到！请各位村民前往广场聚餐！"

小满： [端着一大锅菜跑过来]
         "来来来！今天我做了好多菜！都是地里刚摘的！"

秋棠： [搬着一坛酒]
         "这是我新酿的果酒，大家尝尝！"

阿茶： [端着茶壶]
         "还有我的新茶，大家配着喝，解腻！"

林风： [端着一罐蜂蜜]
         "来来来！蜂蜜配面包，可好吃了！"

【场景】全村人围坐在一起吃饭聊天

老槐： "看着大家热热闹闹的，我心里也暖和。"
        "这就是山庄最好的模样。"

小满： "来来来！大家多吃点！不够我再去端！"

杏儿： "我今天摘了好多果子！大家尝尝！"

[气氛温馨，玩家可以和各角色对话]

与阿茶对话：
  阿茶： "今天的菜真丰盛，谢谢你帮忙修缮主屋。"

与老槐对话：
  老槐： "年轻人啊，山庄交给你，我放心。"

与晚星对话：
  晚星： "今天人真多，星星好像也在为我们高兴呢。"
```

```
【邀请阿茶】
  阿茶： "你邀请我一起活动？太好了！"

  [两人一起去参观主屋]
  阿茶： "你修的主屋，真漂亮。"

  [两人一起去农田]
  小满： "你看，田里的小芽都冒出来了！"
         "今年一定是个丰收年！"

【邀请晚星】
  晚星： "你邀请我？……谢谢你。"
         "我们一起去观星台吧，春天的星星很美。"

  [两人一起去观星台]
  晚星： "你看，春天的星星和冬天不一样。"
         "更柔和，更温暖……就像今天的日子。"

【邀请素锦】
  素锦： "你邀请我？太好了！"
         "我们一起去布置剩下的地方吧！"

  [两人一起装饰]
  素锦： "你看，挂上这个灯笼，是不是更喜庆了？"
         "和你在一起，做什么都开心。"
```

#### 傍晚（17:00-19:00）节日庆典

**[场景]** 山庄主屋前

```
【通知】"山庄苏醒祭庆典即将开始！请各位村民前往主屋前集合！"

老槐： [站在主屋前，看着焕然一新的房子]
         "好！大家都到齐了！"

        "现在，我宣布——主屋修缮完成！"

[村民们鼓掌欢呼]

老槐： "感谢大家今天的帮忙！"
        "这才是山庄应该有的样子——大家一起，热闹温馨！"

雾川： [站在一旁，微微点头]
         "……不错。"

阿巳： [从树上跳下来]
         "哇！好热闹！下次我也要参加！"

【场景】放烟火

清荷： [看着烟火，眼中带着感动]
         "真美……就像荷塘里的萤火虫。"
```

#### 夜间（19:00-22:00）节日尾声

**[场景]** 山庄广场

```
老槐： "今天大家都辛苦了。"

素锦： "今天和你在一起很开心……下次还能一起活动吗？"

林风： "兄弟！今天太开心了！下次一起喝蜂蜜酒！"
```

## 02_灵茶初芽节

### 节日概述

- **序号**：2
- **季节**：春季
- **触发条件**：春季第2周，浓云海
- **持续时间**：全天
- **主要奖励**：初雾绿茶种子×10、茶九专属CG
- **绑定系统**：灵茶系统、角色CG线

---

### 节日流程

#### 清晨（6:00-8:00）开场

**[场景]** 茶田

```
【通知】"灵茶初芽节开始了！今年的浓云海是采茶的好时机！"

阿茶： [站在茶田边，看着云雾缭绕的茶树]
         "今年的新芽发得真好……云海这么浓，茶叶品质肯定特别好。"

茶九： [走过来，表情认真]
         "今天是灵茶初芽节，全山庄的人都可以来采茶。"

茶九： "但是要记住——只采一芽一叶，不能多也不能少。"

阿茶： "而且要趁云海浓的时候采，过了中午云海就散了。"

→ 玩家获得采茶工具
```

#### 上午（8:00-11:00）采茶竞赛

**[场景]** 茶田

```
【通知】"灵茶初芽节采茶竞赛开始！请各位玩家在云海散前尽可能多地采集优质茶叶！"

阿茶： [递给你一个小竹篮]
         "来，用这个装茶叶。"
         "记住，一芽一叶！"

◆ 玩家选择采茶区域

【选项A】茶田中心区（高品质茶叶，但难度较高）
  阿茶： "这里茶叶最好，但也要最细心。"
         "加油！"

【选项B】茶田边缘区（普通品质，容易采集）
  小满： "我来帮你！我以前帮阿茶采过！"
         "虽然不是专业的，但我会努力的！"

[玩家开始采茶]

【小游戏】在云海散去之前，尽可能多地采集茶叶

[采集过程中，触发各种角色的互动]

清荷： [撑着竹筏从旁边路过]
         "加油呀！我来给你们加油！"

桃夭： [跑过来]
         "哇！茶叶好香！"

[小游戏结束]

阿茶： "时间到了！让我看看你采了多少。"

阿茶： [检查竹篮]
         "哇！采了这么多！而且品质都很好！"

阿茶： "你学得好快呀……比当年的我还厉害呢。"
```

#### 中午（11:00-13:00）制茶体验

**[场景]** 茶田的工坊

```
茶九： "采完茶了，现在来学制茶。"

茶九： "灵茶的制作有四道工序：杀青、揉捻、发酵、干燥。"

茶九： "每一步都要用心，火候、时间都要恰到好处。"

◆ 玩家选择学习哪种灵茶的制作

【选项A】初雾绿茶
  阿茶： "初雾绿茶是最基础的茶，我来教你。"

【选项B】云芽茶（需要好感>=4心）
  茶九： "云芽茶需要更高的技巧，我来教你。"
         "但只有达到一定好感的人才能学。"

◆ 玩家选择制茶助手

【邀请阿茶】
  阿茶： "我来帮你揉茶，你负责炒制。"

  [两人一起制茶]
  阿茶： "你学得好认真……和你一起制茶，真开心。"

【邀请茶九】
  茶九： "我来做示范，你来跟着学。"

  [茶九耐心指导]
  茶九： "……你的悟性很高。"
         "比我当年学的时候还要快。"

[制茶完成]

茶九： "好，茶做好了。"

茶九： [品鉴玩家制作的茶]
         "不错，品质很好。"

阿茶： [也尝了一口，眼中带着惊喜]
         "好喝……比我想象的要好喝得多！"
```

#### 下午（13:00-17:00）茶会品鉴

**[场景]** 茶田的观景台

```
【通知】"灵茶初芽节茶会开始！请各位品鉴今年的新茶！"

老槐： "今年的茶会特别热闹啊。"

茶九： [站在台上]
         "欢迎大家参加灵茶初芽节的茶会！"

茶九： "今天大家采的茶，都已经制成了成品。"
        "现在，请大家品鉴！"

【场景】所有人围坐在一起品茶

茶九： [泡茶，展示给大家]
         "这是今年最好的灵茶——用清晨第一缕云海中的茶叶制成。"

老槐： [品了一口]
         "好茶……比我年轻时候喝过的还要好。"

阿茶： [看着玩家]
         "这里面也有你的一份功劳呢。"

锦时： [诗兴大发]
         "云海初芽春意浓，灵茶一盏映山青……"

◆ 玩家可以选择把茶送给谁

【送给阿茶】
  阿茶： [接过茶，脸红了红]
         "这是……给我的？"
         "谢谢你……我会好好珍藏的。"

【送给茶九】
  茶九： [接过茶，点头]
         "你的心意我收下了。"
         "这茶……我会记住的。"

【送给老槐】
  老槐： [笑着接过]
         "好孩子，有心了。"
         "这茶喝下去，心里暖洋洋的。"

【送给锦时】
  锦时： [眼睛一亮]
         "这是给我的？太好了！"
         "我要把它写进诗里！"
```

#### 傍晚（17:00-19:00）云海品茶

**[场景]** 茶田边的观景台

```
[夕阳西下，云海在山间翻涌]

阿茶： "你看，云海要散了。"

阿茶： [和玩家一起看着云海]
         "但今年的云海特别美……可能是因为有你在。"

阿茶： [轻轻地靠在玩家肩膀上]
         "谢谢你陪我过灵茶初芽节。"
         "这是我过得最开心的一次。"

茶九： [走过来，表情比平时柔和]
         "……今天的茶会，你表现得很好。"

茶九： [递过一小包茶叶]
         "这是今年最好的茶，送给你。"

茶九： "……下次再一起制茶吧。"
```

#### 夜间（19:00-22:00）节日尾声

**[场景]** 茶田

```
【通知】"灵茶初芽节即将结束！"

阿茶： "云海散了……但明年的云海还会再来的。"

阿茶： "就像……就像你一样，每年都会来陪我看云海。"

茶九： "明年灵茶初芽节，我还会教你新的制茶方法。"

茶九： "……你要来。"

【奖励发放】
【初雾绿茶种子×10】—— 用于种植灵茶

【茶九好感+30】—— 如果参与了制茶过程
【阿茶好感+20】—— 如果参与了采茶过程

【状态变化】
- 解锁灵茶种植系统

- 节日完成标记
```

- **绑定系统**：灵兽系统、灵气成长线

---

### 节日流程

#### 清晨（6:00-8:00）开场

**[场景]** 农田

```
【通知】"春耕祈福开始！请各位带着灵兽前往农田集合！"

小满： [站在田边，很兴奋]
         "今天是春耕祈福！一年中最重要的日子之一！"

小满： "今年的收成好不好，就看今天了！"

老槐： [走过来]
         "年轻人，今天你要带着灵兽一起耕地。"

老槐： "灵兽可以帮助翻土，让土地更肥沃。"

老槐： "而且，灵兽翻过的地，灵气会更足。"

→ 玩家选择要带的灵兽

【选择云狐】
  云狐： [跳到玩家肩膀上]
          "吱吱！"

【选择雾鹿】
  雾鹿： [轻轻蹭了蹭玩家]
          "……"

【选择竹灵】
  竹灵： [在地上挖了挖]
          "沙沙！"
```

#### 上午（8:00-11:00）灵兽耕地

**[场景]** 农田

```
小满： "好了！开始耕地！"

[灵兽开始帮忙翻土]

小满： [一边干活一边指导]
         "你看，灵兽翻过的地就是不一样！"
         "土都变得又松又软！"

小满： [和玩家一起播种]
         "来，把种子撒在这里……"
         "对，就是这样！"

◆ 玩家选择帮忙方式

【选项A】亲自耕地
  玩家： [拿起锄头开始翻土]

  小满： "哇！你干活好利索！"
         "比我还快！"

【选项B】照顾灵兽
```
