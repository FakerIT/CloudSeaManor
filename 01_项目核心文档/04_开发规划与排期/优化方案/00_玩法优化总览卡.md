# 玩法优化总览卡

## 目标
对齐《星露谷物语》的核心体验：**简单上瘾循环、无压力成长、系统互相增益**。
保留《云海山庄》的核心差异化：**云海生态、灵茶文化、灵兽陪伴、山庄修缮**。

## 三大核心优势（保留并强化）

| 优势 | 设计目的 |
|------|----------|
| 云海生态 | 用"云海浓度+大潮日"替代传统四季天气，提供可预测的策略感 |
| 灵茶文化 | 以"灵茶种植→加工→送礼"为主价值链，不只是卖原料 |
| 灵兽陪伴 | 灵兽作为系统间连接器，提供协助、陪伴与长期成长 |

## 三大优先问题（按阶段解决）

### P0 问题（第1-2阶段解决）
1. **每日循环清晰度不足** - 玩家不知道每天应该做什么
   - 解决：云海预报 + 每日推荐三任务 + 手账提示

2. **系统互连性弱** - 升级收益分散，缺乏全局联动
   - 解决：建筑升级全局联动收益 + 云海守护者契约串联

### P1 问题（第3-4阶段解决）
3. **无压力表达不够明确** - 体力惩罚、错过惩罚让玩家焦虑
   - 解决：轻松模式开关 + 无战斗模式 + 14天保护期

4. **内容解锁节奏混乱** - 作物/角色/节日一次性全开
   - 解决：按章节渐进解锁 + 灵茶三层分级

### P2 问题（第5-6阶段解决）
5. **长期留存目标缺失** - 中后期缺乏明确目标
   - 解决：云海守护者契约6卷系统 + 完美山庄评价

6. **婚后内容空洞** - 结婚后内容减少导致流失
   - 解决：婚后夫妻协助 + 家庭任务 + 共居反馈

## 系统互连设计

```
云海状态 ──→ 作物品质 ──→ 灵茶价值 ──→ 加工收入 ──→ 山庄升级
    │              │              │              │
    ↓              ↓              ↓              ↓
灵界探索 ←── 战斗材料 ←── 装备强化 ←── 守护契约 ←── 长期目标
    │
    ↓
灵兽协助 ←── 社交好感 ←── 送礼偏好 ←── 节日事件
```

## 开发优先级分层

### 第1-2阶段（核心引擎）
- 渲染管线、地图系统、角色移动、帧动画

### 第3-4阶段（数据驱动）
- 物品系统、地块状态机、时间存档、农业闭环

### 第5-6阶段（系统扩展）
- NPC日程、对话引擎、社交好感、工坊加工

### 第7-8阶段（探索特色）
- 云海Shader、灵界探索、战斗系统、采集系统

### 第9-10阶段（体验完善）
- UI框架、音频系统、商店经济、手账日历

### 第11-12阶段（打磨发布）
- 性能优化、细节润色、契约系统、评价系统

## 验收标准

| 指标 | 标准 |
|------|------|
| 新手前7天 | 可稳定完成"推荐3任务"闭环 |
| 系统互连 | 每次建筑升级至少影响2个以上系统 |
| 无压力 | 玩家可在设置中显式选择"自由模式/轻松模式" |
| 长期留存 | 中后期有云海守护者契约作为明确目标 |
