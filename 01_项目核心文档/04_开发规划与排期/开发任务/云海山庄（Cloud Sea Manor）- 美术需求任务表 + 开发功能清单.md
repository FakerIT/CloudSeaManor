# 云海山庄（Cloud Sea Manor）\- 美术需求任务表 \+ 开发功能清单

# 项目基础信息（同步更新）

项目名称：云海山庄 / Cloud Sea Manor

核心定位：治愈系生活模拟 \+ 轻度奇幻冒险 \+ 农场经营

目标平台：PC（Steam 优先）/ Nintendo Switch

技术栈：C\+\+20 / SFML 3\.0\.2 / CMake

当前版本：0\.1\.0（MVP 原型阶段）

核心玩法循环：查看云海 → 照料农田（灵茶） → 采集探索（灵界遗迹） → 社交送礼 → 工坊制作（制茶） → 存档休息

备注：所有任务/功能均适配项目现有架构（app/engine/domain/infrastructure），贴合“云雾缭绕、灵气守护”的治愈奇幻风格，严格遵循项目美术资源规范（存于 docs/ENGINEERING\_STANDARDS\.md）。

# 二、开发功能清单（按开发顺序\+关联美术任务\+项目现有状态排序）

## 模块1：基础引擎与窗口框架（优先开发，适配现有技术栈）

- 功能1\.1：基于SFML 3\.0\.2，确认游戏基础分辨率（1280×720），固定像素对齐，避免模糊，适配PC/Switch双平台渲染差异

- 功能1\.2：基于项目现有app/engine层架构，搭建UI基础框架，导入美术ART\-001的边框、背景、按钮资源，实现窗口打开/关闭基础逻辑，适配SFML事件循环

- 功能1\.4：适配项目CMake构建规则，确保美术资源加载路径正确（优先读取assets目录），与SFML依赖查找逻辑（环境变量→项目同级→报错）保持一致
