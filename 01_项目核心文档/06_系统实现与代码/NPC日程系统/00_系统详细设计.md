## 一、系统概述

### 1.1 核心理念

NPC 日程系统（NpcScheduleSystem）负责管理所有 NPC 角色在全天的移动路线和位置。它从外部数据文件（CSV）加载每个 NPC 的一天日程安排，并根据当前游戏时间（分钟数）实时更新 NPC 的位置和活动状态。

### 1.2 与其他系统的关系

```
NpcScheduleSystem (engine/systems)
    ├── 读取: GameWorldState (world_state.GetClock(), world_state.GetNpcs())
    ├── 写入: NpcActor.shape.setPosition(), NpcActor.visible, NpcActor.state
    ├── 依赖: GameAppNpc.hpp (AnchorForLocation 锚点位置映射)
    ├── 驱动: GameRuntime::UpdateNpcs() 每帧调用
    └── 配合: NpcDialogueManager (对话), DynamicLifeSystem (阶段影响)
```

### 1.3 设计目标

| 目标 | 说明 |
|------|------|
| 数据驱动 | 所有日程从 `Schedule_Data.csv` 加载，策划可独立编辑 |
| 位置映射 | NPC 锚点名称 → 世界坐标的映射表 |
| 夜间隐藏 | 22:00 - 06:00 NPC 不可见 |
| 好感可视化 | NPC 透明度随好感度从 120/210/120 (灰) → 230/180/200 (粉) 渐变 |
| 平滑移动 | NPC 以固定速度向目标锚点移动（55px/s）|

### 2.1 NpcActor 日程结构（在 `GameAppRuntimeTypes.hpp`）

```cpp
struct NpcActor {
    std::string id;                     // NPC 唯一 ID (如 "lin", "bo")
    std::string display_name;           // 显示名称（如 "茶师阿岚"）
    sf::RectangleShape shape;           // SFML 渲染形状
    std::vector<NpcScheduleEntry> schedule;  // 一天日程条目列表
    std::string current_location;      // 当前锚点名称
    std::string current_activity;       // 当前活动
    int favor = 0;                    // 好感度（范围约 -20 ~ +100）
    int heart_level = 0;              // 心级（0-10）
    bool daily_gifted = false;         // 今日已送礼
    bool daily_talked = false;       // 今日已对话
    NpcState state = NpcState::Patrol;  // NPC 状态机
    bool visible = true;              // 是否可见
};

struct NpcScheduleEntry {
    int start_minutes;                // 开始时间（分钟，如 360 = 06:00）
    int end_minutes;                 // 结束时间（分钟，如 720 = 12:00）
    std::string location;             // 锚点名称（对应 AnchorForLocation 的键）
    std::string activity;            // 活动描述（如 "采茶", "休息"）
};
```

## 四、核心算法

### 4.1 每帧更新流程

```
GameRuntime::UpdateNpcs(delta_seconds)
    ↓
NpcScheduleSystem::Update(world_state, delta_seconds)
    ├── for each NpcActor in world_state.GetNpcs()
    │   ├── npc.state = NpcState::Patrol
    │   ├── current_minute = clock.Hour() * 60 + clock.Minute()
    │   ├── UpdateNpcPosition_(npc, current_minute, delta_seconds)
    │   │   ├── if 22:00-06:00: npc.visible = false; return
    │   │   ├── npc.visible = true
    │   │   ├── 查找日程区间: entry.start <= minute < entry.end
    │   │   ├── 更新 current_location / current_activity
    │   │   ├── target = AnchorForLocation(location)
    │   │   ├── delta = target - current_pos
    │   │   ├── distance = sqrt(delta.x² + delta.y²)
    │   │   ├── if distance > 4: npc.shape.setPosition += (delta/distance) * 55 * delta_seconds
    │   │   └── 更新透明度: alpha = 120 + favor_ratio * 110
```

### 4.2 好感度可视化

```
favor_ratio = clamp((favor + 20) / 60.0, 0.0, 1.0)
alpha = 120 + favor_ratio * 110   → 120(陌生) ~ 230(高好感)
R = 140 + favor_ratio * 90     → 140(灰) ~ 230(粉)
G = 130 + favor_ratio * 50     → 130(灰) ~ 180(粉)
B = 120 + favor_ratio * 80     → 120(灰) ~ 200(粉)
```

### 4.3 日程查找算法

```cpp
// 线性查找匹配的日程区间
for (const auto& entry : npc.schedule) {
    if (current_minute >= entry.start_minutes && current_minute < entry.end_minutes) {
        npc.current_location = entry.location;
        npc.current_activity = entry.activity;
        break;  // 第一个匹配即为当前日程
    }
}
```

## 五、函数接口规范

### 5.1 NpcScheduleSystem 头文件（`engine/systems/NpcScheduleSystem.hpp`）

```cpp
class NpcScheduleSystem {
public:
    // 加载日程表（从 CSV 文件）
    // @param csv_path 路径
    // @return 是否加载成功
    // @note 目前为占位函数，始终返回 true
    bool LoadSchedule(const std::string& csv_path);

    // 每帧更新 NPC 位置和状态
    // @param world_state 世界状态引用
    // @param delta_seconds 帧间隔秒数
    void Update(GameWorldState& world_state, float delta_seconds);

    // 获取指定 NPC 的当前位置锚点
    // @param npc_id NPC ID
    // @return 锚点名称，未知返回 ""
    [[nodiscard]] std::string GetCurrentLocation(const std::string& npc_id) const;

    // 获取指定 NPC 的当前活动
    // @param npc_id NPC ID
    // @return 活动描述，未知返回 ""
    [[nodiscard]] std::string GetCurrentActivity(const std::string& npc_id) const;

    // 获取指定 NPC 是否可见
    [[nodiscard]] bool IsVisible(const std::string& npc_id) const;

private:
    void UpdateNpcPosition_(NpcActor& npc, int current_day, int current_minute, float delta_seconds);
};
```
