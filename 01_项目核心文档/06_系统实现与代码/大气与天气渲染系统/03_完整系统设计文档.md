## 一、系统设计概述

### 1.1 核心理念

大气与天气视觉系统负责将 `CloudSystem` 的云海状态和 `GameClock` 的昼夜时段，转化为玩家眼中可感知的天空颜色、粒子特效和光晕氛围。

本系统的核心设计原则：

- **氛围即叙事**：天空颜色和粒子效果传递情感和事件预告，而非装饰
- **数据驱动**：所有视觉参数（颜色、速度、密度）外部配置，无需硬编码
- **零性能惩罚**：对象池粒子、最多 3 次 DrawCall、60 FPS
- **无缝衔接**：复用现有 `ui_layout.json` 的背景色和光晕色

### 1.2 与现有系统的关系

```
CloudSystem (domain) ──────→ AtmosphereDomain (domain)
GameClock (domain)  ──────→ AtmosphereDomain (domain)
                                   │
                                   ▼
                          AtmosphereRenderer (engine)
                                   │
                                   ▼
                          sf::RenderWindow (渲染)
```

| 已有系统 | 提供的输入 | 本系统的输出 |
|---------|-----------|-------------|
| `CloudSystem` | `CloudState`（Clear/Mist/Dense/Tide） | 天空颜色、粒子类型/密度、光晕透明度 |
| `GameClock` | `DayPhase`（Morning/Afternoon/Evening/Night） | 天空颜色、雾效密度、昼夜插值 |
| `UiLayoutConfig` | `aura` 颜色配置 | 大气光晕叠加颜色 |

### 1.3 渲染层次（6层）

| 层次 | 名称 | 说明 |
|------|------|------|
| L0 | 窗口清屏 | `sf::RenderWindow::clear(sf::Color::Black)` 安全网 |
| L1 | 天空渐变 | 全屏双三角渐变，顶部→底部颜色插值 |
| L2 | 远景雾效 | 场景边缘深度雾（shader，可选） |
| L3 | 天气粒子 | 对象池粒子，最多 200 个 |
| L4 | 游戏场景 | `WorldRenderer` 渲染，已有逻辑不变 |
| L5 | 大气光晕 | 全屏半透明叠加，颜色来自 `ui_layout.json` |
| L6 | 调试面板 | `F8` 快捷键切换 |

### 2.4 过渡动画算法

当云海状态或昼夜时段发生变化时：

```
1. 保存当前颜色为 from_
2. 设置目标颜色为 to_
3. 设置 duration = 3.0s，elapsed = 0.0s，transitioning = true
4. 每帧 Update(delta)：
     elapsed += delta
     t = clamp(elapsed / duration, 0, 1)
     eased_t = smoothstep(t) = t²(3 - 2t)   // 缓动曲线
     颜色 = lerp(from, to, eased_t)
5. t >= 1.0 时 transitioning = false
```

### 3.2 粒子系统设计

**对象池**：`std::array<AtmosphereParticle, 200>`，无堆分配。

```cpp
struct AtmosphereParticle {
    sf::Vector2f position;   // 世界坐标
    sf::Vector2f velocity;  // 速度向量
    float size;             // 尺寸（像素）
    float lifetime;          // 总存活时间
    float age;              // 已存活时间
    sf::Color color;        // 当前颜色
    bool active;            // 是否活跃
};
```

**粒子生成策略**：

- 根据 `WeatherProfile.particle_spawn_rate` 定时生成
- 方向：`ParticleDirection`（Left / Right / Radial / Down）
- 颜色：使用 `color_start` → `color_end` 随 `age_ratio` 插值
- 透明度：随 `age_ratio` 衰减，接近寿命末期时淡出

### 5.2 关键代码片段

**AtmosphereDomain 过渡动画**：

```cpp
void AtmosphereDomain::Update(float delta_seconds) {
    if (!is_transitioning_) return;

    transition_elapsed_ += delta_seconds;
    const float t = std::clamp(transition_elapsed_ / transition_duration_, 0.0f, 1.0f);
    const float eased_t = t * t * (3.0f - 2.0f * t);  // smoothstep

    // 颜色插值
    sky_top_r_ = static_cast<int>(std::lerp(static_cast<float>(from_sky_top_r_),
                                             static_cast<float>(to_sky_top_r_), eased_t));
    // ... 其他通道类似

    if (t >= 1.0f) {
        is_transitioning_ = false;
    }
}
```

**AtmosphereRenderer 粒子系统**：

```cpp
void AtmosphereRenderer::UpdateParticleSpawn_(float delta_seconds) {
    if (current_spawn_rate_ <= 0.0f) return;

    particle_spawn_timer_ += delta_seconds;
    const float spawn_interval = 1.0f / current_spawn_rate_;

    while (particle_spawn_timer_ >= spawn_interval
           && active_particle_count_ < static_cast<std::size_t>(current_max_particles_)) {
        AtmosphereParticle* p = AcquireParticle_();
        if (!p) break;
        p->position = GetSpawnPosition_(current_direction_, ...);
        p->velocity = GetMoveDirection_(current_direction_, ...) * current_speed_;
        p->size = RandFloat(rng_, size_min_, size_max_);
        p->lifetime = RandFloat(rng_, lifetime_min_, lifetime_max_);
        p->age = 0.0f;
        particle_spawn_timer_ -= spawn_interval;
        ++active_particle_count_;
    }
}
```

### 9.1 远景雾效 Shader（L2）

使用 GLSL Fragment Shader 实现屏幕空间雾效，增强深度感：

```glsl
// fog.glsl
uniform sampler2D sceneTexture;
uniform vec3 fog_color;
uniform float fog_density;
uniform float camera_depth;

void main() {
    vec4 color = texture2D(sceneTexture, gl_TexCoord[0].xy);
    float depth = gl_FragCoord.z / gl_FragCoord.w;
    float fog_factor = 1.0 - exp(-fog_density * depth);
    color.rgb = mix(color.rgb, fog_color, fog_factor);
    gl_FragColor = color;
}
```

### 9.2 季节性天空扩展

当前 `atmosphere_profiles.csv` 以 Clear/Mist/Dense/Tide × Morning/Afternoon/Evening/Night 为主。
后续可扩展按季节（春/夏/秋/冬）进一步细分的的天空配置，通过 `sort_order` 优先级实现覆盖。
