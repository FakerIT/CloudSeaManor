# 大气与天气渲染系统 - 主循环集成指南

> 本文档说明如何在 `GameApp` 主循环中集成 AtmosphereRenderer。

---

## 一、GameApp 中的成员变量

在 `GameApp.hpp` 中添加以下成员：

```cpp
// 新增：大气渲染器
std::unique_ptr<CloudSeamanor::engine::AtmosphereRenderer> atmosphere_renderer_;
CloudSeamanor::domain::AtmosphereDomain atmosphere_domain_;
```

---

## 二、GameApp::Run() 初始化阶段

在 `GameApp::Run()` 中，资源加载完成后添加：

```cpp
// 在 ui_system_->InitializePanels(...) 之后添加：

// 初始化大气系统
{
    CloudSeamanor::infrastructure::AtmosphereDataLoader loader;
    std::vector<CloudSeamanor::domain::SkyProfile> sky_profiles;
    std::vector<CloudSeamanor::domain::WeatherProfile> weather_profiles;

    const std::filesystem::path data_dir = "assets/data/atmosphere";
    loader.LoadWithDefaults(data_dir, sky_profiles, weather_profiles);

    atmosphere_domain_.Initialize(sky_profiles, weather_profiles);
    atmosphere_domain_.ForceState(
        runtime_.Systems().GetCloud().CurrentState(),
        runtime_.WorldState().GetClock().CurrentPhase(),
        true
    );

    atmosphere_renderer_ = std::make_unique<CloudSeamanor::engine::AtmosphereRenderer>();
    atmosphere_renderer_->Initialize(window, atmosphere_domain_, ui_layout_config_);
}
```

---

## 三、GameApp::Update() 更新阶段

在 `GameApp::Update()` 中，在主循环更新之前添加：

```cpp
void GameApp::Update(float delta_seconds) {
    if (show_main_menu_) {
        return;
    }
    auto& ws = runtime_.WorldState();
    auto& sys = runtime_.Systems();

    // 更新大气系统
    atmosphere_domain_.SyncFromCloudAndClock(
        sys.GetCloud(),
        ws.GetClock()
    );
    atmosphere_domain_.Update(delta_seconds);
    atmosphere_renderer_->Update(delta_seconds);

    // ... 原有更新逻辑 ...
}
```

---

## 四、GameApp::Render() 渲染阶段

在 `GameApp::Render()` 中，用大气渲染替代现有的 `window.clear()` 背景色：

```cpp
void GameApp::Render(sf::RenderWindow& window) {
    auto& ws = runtime_.WorldState();

    // 使用大气天空渐变替代纯色背景清屏
    window.clear(sf::Color::Black);  // 清屏为黑色作为安全网

    if (show_main_menu_) {
        RenderMainMenu_(window);
        window.display();
        return;
    }

    // 渲染天空渐变和粒子（大气层 L1 + L3）
    atmosphere_renderer_->DrawSkyGradient(window);
    atmosphere_renderer_->DrawWeatherParticles(window);

    // 渲染游戏场景（L3 场景层）
    world_renderer_.Render(window, ws);

    // 渲染 HUD 和 UI
    pixel_hud_->Render(window);
    if (hud_renderer_) {
        hud_renderer_->Render(window, ws);
    } else {
        UiRenderOptions opts;
        opts.show_debug = ws.GetTutorial().show_debug_overlay;
        opts.show_level_up = ws.GetLevelUpOverlayActive();
        opts.show_festival = true;
        ui_system_->Render(window, opts);
    }

    // 渲染大气光晕叠加（L5）
    atmosphere_renderer_->DrawAuraOverlay(window);

    // 渲染调试面板
    atmosphere_renderer_->DrawDebugPanel(window);

    window.display();
}
```

**注意**：`BgColor_` 和 `AuraColor_` 的逻辑已被大气系统替代，旧的 `window.clear(BgColor_(...))` 可以移除。

## 五、GameApp::ProcessEvents() 快捷键

在键盘事件处理中添加大气调试快捷键：

```cpp
// 在 HandleGameAction 或单独的键盘处理中添加：
if (const auto* key = ev->getIf<sf::Event::KeyPressed>()) {
    switch (key->scancode) {
    case sf::Keyboard::Scancode::F8:
        atmosphere_renderer_->ToggleDebugPanel();
        break;
    case sf::Keyboard::Scancode::Num1:
        runtime_.Systems().GetCloud().CycleDebugState();  // Clear
        atmosphere_domain_.ForceState(
            runtime_.Systems().GetCloud().CurrentState(),
            runtime_.WorldState().GetClock().CurrentPhase(), false);
        break;
    case sf::Keyboard::Scancode::Num2:
        runtime_.Systems().GetCloud().SetStates(
            CloudSeamanor::domain::CloudState::Mist,
            runtime_.Systems().GetCloud().ForecastState());
        atmosphere_domain_.ForceState(
            CloudSeamanor::domain::CloudState::Mist,
            runtime_.WorldState().GetClock().CurrentPhase(), false);
        break;
    case sf::Keyboard::Scancode::Num3:
        runtime_.Systems().GetCloud().SetStates(
            CloudSeamanor::domain::CloudState::DenseCloud,
            runtime_.Systems().GetCloud().ForecastState());
        atmosphere_domain_.ForceState(
            CloudSeamanor::domain::CloudState::DenseCloud,
            runtime_.WorldState().GetClock().CurrentPhase(), false);
        break;
    case sf::Keyboard::Scancode::Num4:
        runtime_.Systems().GetCloud().ForceTide();
        atmosphere_domain_.ForceState(
            CloudSeamanor::domain::CloudState::Tide,
            runtime_.WorldState().GetClock().CurrentPhase(), false);
        break;
    case sf::Keyboard::Scancode::T:
        // 触发动画过渡测试（强制切换到下一个时段）
        {
            using Phase = CloudSeamanor::domain::DayPhase;
            const auto current = runtime_.WorldState().GetClock().CurrentPhase();
            Phase next = Phase::Morning;
            switch (current) {
            case Phase::Morning:    next = Phase::Afternoon; break;
            case Phase::Afternoon: next = Phase::Evening;   break;
            case Phase::Evening:   next = Phase::Night;     break;
            case Phase::Night:     next = Phase::Morning;    break;
            }
            atmosphere_domain_.ForceState(
                runtime_.Systems().GetCloud().CurrentState(), next, false);
        }
        break;
    case sf::Keyboard::Scancode::R:
        atmosphere_renderer_->ClearDebugWeatherOverride();
        atmosphere_domain_.ForceState(
            runtime_.Systems().GetCloud().CurrentState(),
            runtime_.WorldState().GetClock().CurrentPhase(), true);
        break;
    case sf::Keyboard::Scancode::P:
        atmosphere_renderer_->SetPaused(!atmosphere_renderer_->IsPaused());
        break;
    case sf::Keyboard::Scancode::Equal:
    case sf::Keyboard::Scancode::Add:
        atmosphere_renderer_->SetParticleDensityMultiplier(
            atmosphere_renderer_->GetParticleDensityMultiplier() + 0.1f);
        break;
    case sf::Keyboard::Scancode::Hyphen:
    case sf::Keyboard::Scancode::Subtract:
        atmosphere_renderer_->SetParticleDensityMultiplier(
            atmosphere_renderer_->GetParticleDensityMultiplier() - 0.1f);
        break;
    }
}
```

## 六、存档集成

在 `GameAppSave.cpp` 的存档/读档流程中添加大气状态：

### 保存
```cpp
// 在存档文件末尾追加一行
oss << atmosphere_domain_.SaveState() << "\n";
```

### 读取
```cpp
// 在读取时解析
if (line.rfind("atmosphere|", 0) == 0) {
    atmosphere_domain_.LoadState(line);
}
```
