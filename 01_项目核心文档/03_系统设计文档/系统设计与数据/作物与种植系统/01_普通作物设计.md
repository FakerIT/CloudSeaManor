**版本**：1.0

## 1. 普通作物总览（18种）

### 1.1 设计原则

- **饱食恢复**：所有普通作物产出的食材均可食用/烹饪，恢复体力
- **前期主力**：Year 1 春季的核心收入来源，稳定、低门槛
- **季节特色**：每个季节 4~5 种作物，各有定位（高产/速生/高值/加工原料）
- **稀有补充**：夏秋各有 1~2 种稀有作物，提供惊喜感

### 1.2 作物分类

| 分类 | 说明 | 作物示例 |
|------|------|---------|
| **饱腹主食** | 大量饱食恢复，游戏前期保命 | 云雾米、薯类、根茎类 |
| **速生作物** | 生长周期短（7~9天），快速迭代 | 萝卜、生菜、豆荚 |
| **经济作物** | 稳定中等收益，加工原料 | 蓝莓、葡萄、番茄 |
| **稀有惊喜** | 低概率高收益，浓云海触发灵化 | 火龙果、茶油果、灵芝草 |
| **季节限定** | 仅特定季节可种，高单价 | 雪莲根、霜叶菜、竹笋 |
| **多功能** | 食用+送礼+加工多用途 | 蜜桃、蓝莓、苹果 |

## 3. 饱食度系统（ Hunger System）

### 3.1 系统定位

> **饱食度是玩家的"生命线"，不是惩罚机制。**  
> 饥饿不会导致游戏失败，而是降低行动效率和体力恢复速度。

### 3.2 饱食度模型

```
饱食度（Hunger）
├── 上限：100 点
├── 消耗来源：
│   ├── 每小时游戏时间：-1 点/小时（被动消耗）
│   ├── 重体力活动（采矿/战斗）：额外 -2 点/次
│   └── 灵茶饮用：不消耗饱食度（但也不恢复）
└── 恢复来源：
    ├── 直接食用食材：+8～+28 点（按食材）
    └── 食用烹饪料理：+15～+50 点（按菜品复杂度）
```

### 3.3 饱食度阈值效果

| 饱食度区间 | 状态 | 效果 |
|-----------|------|------|
| 80~100 | **饱足** | 体力恢复速度 +20%，行动效率正常 |
| 40~79 | **正常** | 无加成无惩罚 |
| 20~39 | **饥饿** | 体力恢复速度 -30%，体力消耗 +15% |
| 0~19 | **空腹** | 体力恢复停止，体力消耗 +30%，视觉效果（屏幕边缘微暗） |
| 0 | **空腹警告** | 每分钟弹出提示，强制指引前往进食 |

### 3.4 饱食度恢复表

| 作物类别 | 代表作物 | 饱食恢复 | 备注 |
|---------|---------|:-------:|------|
| 根茎类 | 白萝卜/霜萝卜/红薯 | +15~+18 | 稳定饱腹 |
| 果蔬类 | 番茄/蓝莓/草莓 | +15~+18 | 清爽口感 |
| 主食类 | 云雾米/芋头 | +16~+20 | 大量饱腹 |
| 高能量 | 岩蜜瓜/火龙果 | +25~+28 | 极端饱腹 |
| 轻食类 | 雾灵花/雪绒花 | +8~+10 | 轻微恢复 |
| 菌菇类 | 云耳/冬菇/灵芝草 | +12~+14 | 含灵气加成 |

### 3.5 饱食度实现要点

1. **不设硬性饥饿惩罚**：空腹不会死亡，只是效率下降
2. **食材过剩警告**：当仓库积累大量食材时，HUD 显示"食材库存充足，可考虑出售或加工"
3. **饱食度不存档**：每天睡觉后重置为 80（起床后饱足状态）
4. **早餐机制**：起床后可选"吃早餐"，消耗 1 份食材，恢复到 100

## 4. 季节适种速查

### 4.1 春季适种

```
最佳组合（穷开局）：
  ×15 白萝卜 (6天) → 每轮 3天 → 第一季可种 9轮 → 收入 810G × 9 = 7290G

进阶组合（Year 1 夏秋准备）：
  ×5 云雾米 (12天) → 饱食自给
  ×5 翠竹笋 (8天) → 加工竹韵茶点，送礼+收入
  ×5 草莓 (10天) → 积累本金至 5000G+
```

### 4.2 夏季适种

```
稳定组合：
  ×8 番茄 (12天) → 高产稳收
  ×5 蓝莓 (16天) → 后期爆发收入
  ×3 岩蜜瓜 (20天) → 长期投资，夏末冬初收获

冒险组合：
  ×2 火龙果 (24天) → 灵化变种 = 2~3倍收益
  ×3 金丝南瓜 (14天) → 装饰+少量收入
```

### 4.3 秋季适种

```
丰收组合：
  ×6 葡萄 (13天) → 送礼+酿酒双重价值
  ×5 栗子 (15天) → 高单价，秋季主力
  ×3 茶油果 (22天) → 全年最慢，秋季种最好

稀有组合：
  ×2 灵芝草 (25天) → 高级加工材料，价值极高
  ×3 雾灵花 (8天) → 速生，送礼社交
```

### 4.4 冬季适种

```
温饱组合：
  ×6 羽衣甘蓝 (10天) → 稳定饱食+收入
  ×4 霜萝卜 (12天) → 冬季主力经济
  ×2 雪莲根 (15天) → 抗寒特效，高价值

特殊组合（温室）：
  温室种葡萄/草莓 → 冬季高价出售
```

## 5. 灵化变种机制

> **浓云海日，普通作物有概率触发"灵化变种"，售价 2~3 倍**

### 5.1 触发条件

- 当前天气为 **DenseCloud（浓云海）** 或 **Tide（大潮）**
- 作物处于 **成熟状态**（`ready == true`）
- 随机概率：浓云海 **15%**，大潮 **30%**

### 5.2 灵化变种效果

| 原作物 | 灵化后名称 | 基础卖价增幅 | 特殊效果 |
|--------|----------|:----------:|---------|
| 火龙果 | 灵火龙果 | 3.0× | 附带"火属性护盾"buff |
| 蓝莓 | 灵云莓 | 2.5× | 制作灵药时成功率+20% |
| 栗子 | 灵风栗 | 2.2× | 加工成灵饲料，产量翻倍 |
| 茶油果 | 灵茶油果 | 2.5× | 灵气肥效果+30% |

### 5.3 灵化视觉提示

- 作物地块发光（紫色粒子效果）
- 收获时显示"灵化变种！获得 [物品名]"特殊提示
- 灵化作物在背包中显示紫色光晕

## 7. 与现有代码的接口

### 7.1 CropTable.csv 新增字段

```csv
id,name,seed_item_id,harvest_item_id,growth_time,stages,base_harvest,stamina_cost,tags,hunger_restore,special_effect,season
```

新增字段：
- `hunger_restore`：饱食恢复值（整数）
- `special_effect`：特殊效果描述（字符串）
- `season`：适种季节（spring/summer/autumn/winter/all）

### 7.2 饱食度系统代码落点

| 组件 | 文件路径 | 职责 |
|------|---------|------|
| **HungerSystem** | `domain/HungerSystem.hpp`（新增） | 管理饱食度值、消耗、恢复 |
| **StaminaSystem 扩展** | `domain/Stamina.hpp` | 饱食度影响体力恢复速度的联动 |
| **FarmingResult 扩展** | `engine/FarmingSystem.hpp` | 收获时附带饱食恢复值提示 |
| **HUD 扩展** | `PixelGameHud.hpp` | 饱食度条显示（与体力条并列） |
| **存档扩展** | `SaveSlotManager` | 保存/恢复饱食度状态 |
