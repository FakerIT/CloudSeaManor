## 1. 灵茶定位总览

> **灵茶是《云海山庄》的核心经济与社交系统，是玩家长期投入的动力源。**

### 1.1 三大核心价值

| 价值维度 | 说明 | 变现方式 |
|---------|------|---------|
| **经济价值** | 远高于普通作物，加工链放大 3~5 倍 | 直接出售、加工后出售、游客收购 |
| **社交价值** | 多个角色有灵茶喜好，送礼效率极高 | 送礼好感、剧情解锁、结婚条件 |
| **Buff 价值** | 饮用/加工产出增益道具 | 临时属性提升、战斗减伤、速度加成 |

### 1.2 灵茶 vs 普通作物对比

| 对比维度 | 普通作物 | 灵茶 |
|---------|---------|------|
| **定位** | 前期稳定收入、饱食来源 | 中后期经济支柱 |
| **饱食恢复** | 有（+8~+28） | 无（纯经济/社交） |
| **加工价值** | 基础料理 | 初加工 + 精加工双层链（3~5倍放大） |
| **社交价值** | 部分作物有送礼效果 | 核心送礼系统，多角色最爱 |
| **稀有性** | 随时可种 | 按季节解锁，需云海条件 |
| **品质跨度** | 普通 → 灵品（5级） | 普通 → 圣品（受云海主导） |
| **体力消耗** | 中等（3~10） | 略高（6~12） |
| **种植要求** | 基础地块 | 基础地块 + 茶园地块（后期） |

## 3. 品质系统

### 3.1 五级品质定义

| 品质等级 | 英文 | 售价倍率 | 视觉标识 | 触发条件 |
|---------|------|:-------:|---------|---------|
| 普通（Normal） | Normal | 1.0× | 无特殊标识 | 晴朗 |
| 优质（Fine） | Fine | 1.5× | 浅蓝光晕 | 薄雾（稳定） |
| 稀有（Rare） | Rare | 2.0× | 紫色光晕 | 浓云海（稳定） |
| 灵品（Spirit） | Spirit | 3.0× | 金色光晕 + 粒子 | 大潮（稳定） |
| 圣品（Holy） | Holy | 5.0× | 彩虹光晕 + 特效音 | 大潮 + 高灵气 + 特殊条件 |

### 3.2 品质判定公式

```
quality = Normal
density = CloudSystem.CurrentSpiritDensity()  // 0.0 / 0.3 / 0.7 / 1.0

if density >= 1.0:           // 大潮
    if aura >= 600:
        if tea_garden.has_tea_soul_flower:
            quality = Holy     // 圣品
        else:
            quality = Spirit   // 灵品
    else:
        quality = Rare
elif density >= 0.7:         // 浓云海
    quality = Rare
elif density >= 0.3:         // 薄雾
    if Random() < 0.5:
        quality = Fine
    else:
        quality = Normal
else:                         // 晴朗
    quality = Normal

// 额外加成：施肥
if fertilizer_type == "premium":
    quality += 1 级（上限 Holy）
```
