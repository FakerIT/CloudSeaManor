## 2. CropTable.csv 扩展字段定义

### 2.1 新字段说明

在现有 9 列基础上，新增 4 列：

```csv
id,name,seed_item_id,harvest_item_id,growth_time,stages,base_harvest,stamina_cost,tags,hunger_restore,special_effect_id,season_tag,fertilizer_multiplier
```

| 列名 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `id` | string | 作物唯一标识 | `turnip` |
| `name` | string | 中文展示名 | `白萝卜` |
| `seed_item_id` | string | 种子物品ID | `TurnipSeed` |
| `harvest_item_id` | string | 收获物品ID | `Turnip` |
| `growth_time` | int | 生长时间（秒） | `480`（=8分钟=8天） |
| `stages` | int | 生长阶段数 | `3` |
| `base_harvest` | int | 基础产量 | `3` |
| `stamina_cost` | int | 每次操作体力消耗 | `4` |
| `tags` | string | 标签（分号分隔） | `core;season_spring` |
| `hunger_restore` | int | 饱食恢复值 | `15` |
| `special_effect_id` | string | 特殊效果ID（可选） | `restore_stamina_speed` |
| `season_tag` | string | 适种季节 | `spring` |
| `fertilizer_multiplier` | float | 肥料效果倍率 | `1.0` |

> **growth_time 单位说明**：`growth_time` 字段的单位是**秒**。  
> 在游戏时间系统中：1 秒 = 1 游戏分钟，1 游戏天 = 1440 秒（24×60）。  
> `growth_time=480` 表示 480 秒 = 480 游戏分钟 = 8 游戏天。

### 6.4 HungerSystem（新增领域类）

文件：`include/CloudSeamanor/HungerSystem.hpp`（新增）

```cpp
#pragma once

// ============================================================================
// 【HungerSystem】饱食度管理系统
// ============================================================================
// 负责玩家饱食度（Hunger）的值管理、消耗与恢复。
//
// 主要职责：
// - 管理饱食度当前值和上限（100）
// - 处理饱食度消耗（每小时游戏时间、重体力活动）
// - 处理饱食度恢复（食用食材、烹饪料理）
// - 根据饱食度阈值提供效率修正
//
// 与其他系统的关系：
// - 依赖：Inventory（食材消耗）
// - 被依赖：StaminaSystem（饱食度影响体力恢复速度）
// - 被依赖：PixelGameHud（饱食度条显示）
// - 被依赖：GameAppSave（存档/读档）
// ============================================================================

#include <string>

namespace CloudSeamanor::domain {

class HungerSystem {
public:
    // ========================================================================
    // 【Initialize】初始化
    // ========================================================================
    void Initialize(int current = 80, int max_value = 100);

    // ========================================================================
    // 【Tick】每帧更新（消耗饱食度）
    // ========================================================================
    // @param delta_seconds 距离上次更新的时间（秒）
    void Tick(float delta_seconds);

    // ========================================================================
    // 【Consume】消耗饱食度（负值表示减少）
    // ========================================================================
    void Consume(int amount);

    // ========================================================================
    // 【Restore】恢复饱食度
    // ========================================================================
    void Restore(int amount);

    // ========================================================================
    // 【Refill】起床后重置为满（吃早餐）
    // ========================================================================
    void Refill();

    // ========================================================================
    // 【HasEaten】检查是否已吃过早餐
    // ========================================================================
    bool HasEaten() const { return has_eaten_today_; }

    // ========================================================================
    // 【SetEaten】标记为已吃早餐（睡觉时重置）
    // ========================================================================
    void SetEaten(bool value) { has_eaten_today_ = value; }

    // ========================================================================
    // 【状态查询】
    // ========================================================================
    [[nodiscard]] int Current() const noexcept { return current_; }
    [[nodiscard]] int Max() const noexcept { return max_; }
    [[nodiscard]] float Ratio() const noexcept {
        return max_ > 0 ? static_cast<float>(current_) / max_ : 0.0f;
    }

    [[nodiscard]] HungerState State() const noexcept {
        if (current_ >= 80) return HungerState::Satiated;
        if (current_ >= 40) return HungerState::Normal;
        if (current_ >= 20) return HungerState::Hungry;
        return HungerState::Starving;
    }

    // ========================================================================
    // 【效率修正查询】
    // ========================================================================
    // 饱食度对体力恢复速度和消耗的影响
    [[nodiscard]] float StaminaRecoveryMultiplier() const noexcept {
        switch (State()) {
        case HungerState::Satiated:  return 1.20f;
        case HungerState::Normal:    return 1.00f;
        case HungerState::Hungry:    return 0.70f;
        case HungerState::Starving:  return 0.00f;
        }
        return 1.0f;
    }

    [[nodiscard]] float StaminaCostMultiplier() const noexcept {
        switch (State()) {
        case HungerState::Satiated:  return 1.00f;
        case HungerState::Normal:    return 1.00f;
        case HungerState::Hungry:    return 1.15f;
        case HungerState::Starving:  return 1.30f;
        }
        return 1.0f;
    }

private:
    void ApplyClamp_();

    int current_ = 80;
    int max_ = 100;
    float accumulated_seconds_ = 0.0f;
    bool has_eaten_today_ = false;

    static constexpr int kHungerPerHour = 1;  // 每小时消耗1点饱食度
};

// ============================================================================
// 【HungerState】饱食度状态枚举
// ============================================================================
enum class HungerState {
    Satiated,   // 饱足（80~100）：体力恢复+20%
    Normal,     // 正常（40~79）：无加成
    Hungry,     // 饥饿（20~39）：恢复-30%，消耗+15%
    Starving,   // 空腹（0~19）：恢复停止，消耗+30%
};

} // namespace CloudSeamanor::domain
```
