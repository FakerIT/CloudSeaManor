## 1. 地块生命周期

### 1.1 地块状态机

```
[未开垦] ──(斧/镐/镰)──► [已开垦·裸地]
     │                          │
     │                          ▼
     │                   [已整地] ←─(锄头 E)
     │                     │
     │                     ▼
     │               [已播种·待浇水] ←─(种子 E)
     │                     │
     │                     ▼
     │              [已浇水·生长期]
     │                     │
     │        ┌────────────┴────────────┐
     │        ▼            ▼             ▼
     │   [生长期1]     [生长期2]     [生长期3]
     │        │            │             │
     │        └────────────┴────────────┘
     │                     │
     │                     ▼
     │              [成熟·待采收] ←─(E 收获)
     │                     │
     │                     ▼
     │              [已收获·待整地] ──► [已整地] (循环)
     │
     ▼ (铲除)
[已铲除·恢复裸地]
```

### 1.2 地块操作与消耗

| 操作 | 工具 | 体力消耗 | 前置条件 | 后置状态 |
|------|------|:-------:|---------|---------|
| **开垦（石）** | 斧头 | 8 | 已开垦地块有石块 | cleared = true |
| **开垦（木桩）** | 锄头 | 6 | 已开垦地块有木桩 | cleared = true |
| **开垦（杂草）** | 镰刀 | 3 | 已开垦地块有杂草 | cleared = true |
| **整地** | 锄头（自动） | 5 | cleared = true | tilled = true |
| **播种** | 种子（自动） | 3 | tilled = true + 背包有种子 | seeded = true |
| **浇水** | 水壶（自动） | 2 | seeded = true | watered = true |
| **施肥** | 肥料（手动） | 2 | seeded = true | fertilized = true |
| **采收** | 自动（E交互） | 4 | ready = true | 产出物品 |
| **铲除** | 铲子（手动） | 6 | seeded = true | 恢复裸地 |

### 1.3 障碍物清除规则

| 障碍类型 | 英文ID | 清除工具 | 次数 | 体力消耗 |
|---------|--------|---------|:----:|:-------:|
| 石头 | Stone | 斧头 | 3次 | 8×3 = 24 |
| 木桩 | Stump | 锄头 | 2次 | 6×2 = 12 |
| 杂草 | Weed | 镰刀 | 1次 | 3 |

### 1.4 一键操作链

为了减少重复操作的疲劳感，支持以下快捷操作：

| 操作组合 | 说明 | 触发条件 |
|---------|------|---------|
| **整地+播种** | 锄头→种子，连续操作 | 背包有对应种子 |
| **浇水全部** | 对任意地块按 E，自动浇灌所有干燥地块 | 背包有水壶 |
| **批量采收** | 长按 E 键 0.5 秒，收获所有成熟地块 | 至少1个成熟地块 |
| **一键施肥** | 对任意地块按 E，若有肥料则自动施用 | 背包有肥料 |

## 2. 生长模型

### 2.1 基础生长公式

```
每帧生长量 = delta_seconds × growth_multiplier × cloud_modifier

growth_multiplier =
    base (1.0)
    + fertilizer_bonus      // 普通肥 +0.2 / 灵气肥 +0.45 / 云海精华 +0.6
    + greenhouse_bonus      // 温室 +0.1
    + sprinkler_bonus       // 洒水器 +0.05
    + season_bonus          // 适季 +0.0 / 不适季 -0.5 (温室屏蔽)

cloud_modifier =
    Clear → 1.0
    Mist → 1.15
    DenseCloud → 1.40
    Tide → 1.60
```

### 2.2 阶段计算

```
每个阶段的阈值 = (stage_index) / total_stages

示例：4阶段作物
  阶段0 → 阶段1：growth >= 1/4 = 0.25 → 第1阶段生长
  阶段1 → 阶段2：growth >= 2/4 = 0.50 → 第2阶段生长
  阶段2 → 阶段3：growth >= 3/4 = 0.75 → 第3阶段生长
  阶段3 → 阶段4：growth >= 4/4 = 1.00 → 成熟
```

### 2.3 生长时间速查

| 作物类型 | 基础天数 | 薄雾天数 | 浓云海天数 | 大潮天数 |
|---------|:-------:|:-------:|:---------:|:-------:|
| 普通作物（6~9天） | 6~9天 | 5.2~7.8天 | 4.3~6.4天 | 3.8~5.6天 |
| 普通作物（10~15天） | 10~15天 | 8.7~13天 | 7.1~10.7天 | 6.3~9.4天 |
| 普通作物（16~25天） | 16~25天 | 13.9~21.7天 | 11.4~17.8天 | 10~15.6天 |
| 入门灵茶（8~11天） | 8~11天 | 7~9.5天 | 5.7~7.8天 | 5~6.9天 |
| 中级灵茶（11~16天） | 11~16天 | 9.5~13.9天 | 7.8~11.4天 | 6.9~10天 |
| 高级灵茶（14~18天） | 14~18天 | 12.2~15.6天 | 10~12.8天 | 8.8~11.2天 |
| 传说灵茶（28天） | 28天 | 24.3天 | 20天 | 17.5天 |

### 2.4 日/夜生长规则

- **白天（6:00~22:00）**：正常生长，有阳光
- **夜晚（22:00~6:00）**：生长速度降低至 **50%**（模拟夜间无光合作用）
- **雨季效果（未来扩展）**：额外 +10% 生长速度（自动浇水效果）

## 3. 云海天气对种植的全面影响

### 3.1 天气对地块系统的完整影响矩阵

| 影响维度 | 晴朗（Clear） | 薄雾（Mist） | 浓云海（DenseCloud） | 大潮（Tide） |
|---------|:-----------:|:-----------:|:-----------------:|:-----------:|
| **生长速度** | 1.0× | 1.15× | 1.40× | 1.60× |
| **品质基准** | 普通 | 优质（50%） | 稀有 | 灵品 |
| **每日灵气** | +5 | +12 | +25 | +80 |
| **稀有掉落** | 基础 | +20% | +50% | 100% |
| **土壤湿度** | -1格/天 | 0（保湿） | +1格/天 | +2格/天 |
| **病虫害概率** | 5% | 3% | 4% | 6% |
| **肥料效果** | 正常 | +10% | +20% | +30% |
| **温室效果** | 1.0× | 1.1× | 1.3× | 1.5× |
| **灵化变种概率** | 0% | 0% | 15% | 30% |

### 3.2 土壤湿度模型

```
watered 状态每日重置规则（每天睡觉前）：

if no rain_today:
    if weather == Clear:      watered = false     // 晴天蒸发
    if weather == Mist:      watered = watered    // 薄雾保湿
    if weather == DenseCloud: watered = true     // 浓云保湿
    if weather == Tide:       watered = true       // 大潮保湿
```

### 3.3 天气预报与种植规划

- **22:00 公布次日天气预报**
- 晴朗日：提前浇水，优先安排户外劳作
- 薄雾日：标准运营
- 浓云海日：**最佳种植日**，生长加速 + 稀有品质
- 大潮日：**传说灵茶种植窗口**，提前备好种子和肥料

## 4. 病虫害与枯萎系统

### 4.1 病虫害触发规则

| 事件 | 触发概率 | 触发条件 | 效果 |
|------|:-------:|---------|------|
| **病害** | 3~6%/天 | 潮湿（连续浇水3天+）且无风 | 生长速度 -50%，品质 -1级 |
| **虫害** | 2~5%/天 | 气温极端（浓云海/大潮） | 产量 -30% |
| **枯萎** | 100% | 不适季 + 无温室 | 作物死亡，损失种子成本 |
| **倒伏** | 5%/天 | 风暴日（大潮时低概率触发） | 产量 -50% |

### 4.2 病虫害状态显示

| 状态 | 视觉 | 地块提示 |
|------|------|---------|
| 正常 | 绿色生长指示 | 正常 |
| 病害 | 黄色警告图标 | "地块染病" |
| 虫害 | 红色虫子图标 | "地块有虫" |
| 枯萎 | 灰色枯萎图标 | "作物枯萎" |
| 倒伏 | 倾斜图标 | "作物倒伏" |

### 4.3 病虫害处理

| 事件 | 处理道具 | 成本 | 效果 |
|------|---------|------|------|
| 病害治疗 | 除菌剂（PesticideItem） | 45G | 消除病害，恢复正常生长 |
| 虫害治疗 | 杀虫剂（PesticideItem） | 45G | 消除虫害，恢复产量 |
| 枯萎恢复 | 无（不可逆） | - | 铲除后重新播种 |
| 倒伏恢复 | 无（不可逆） | - | 自然恢复（3天后） |

### 4.4 灵兽协助防治

- **灵兽拥有"病虫防治"特性**时，可每天自动驱赶病虫害
- 灵兽放养在田地时，该田地内病虫害概率降低 **80%**
- 灵兽可派遣到指定地块进行专项防治

## 5. 灌溉系统

### 5.1 洒水器（Sprinkler）

| 属性 | 基础版 | 高级版 | 豪华版 |
|------|:-----:|:------:|:------:|
| **价格** | 180G | 450G | 900G |

| **自动化** | 每日自动 | 每日自动 | 每日自动 |
| **体力消耗** | 0 | 0 | 0 |

### 5.2 洒水器安装与维护

```
安装：购买洒水器 → 对准地块按 E → 安装成功
维护：洒水器到期前1天提示 → 到期后需重新购买安装
效果：覆盖半径内的地块每天自动浇水（watered = true）
```

### 5.3 温室灌溉

- 温室地块自带湿度维持系统，**无需浇水**
- 温室不受天气影响（全自动生长）
- 温室可种植任意季节作物（打破季节限制）

## 6. 施肥系统

### 6.1 肥料类型

| 肥料 | ID | 成本 | 效果 | 持续 |
|------|----|:----:|------|:----:|
| 普通肥 | `basic_fertilizer` | 60G | 生长速度 +20% | 当季 |
| 灵气肥 | `spirit_fertilizer` | 90G | 生长速度 +45% + 品质+1 | 当季 |
| 云海精华肥 | `cloud_essence_fertilizer` | 免费（稀有掉落） | 生长速度 +60% + 品质+2 | 当季 |
| 茶魂花肥 | `tea_soul_fertilizer` | 150G | 生长速度 +50% + 圣品概率+10% | 当季 |

### 6.2 施肥时机

| 时机 | 效果 | 备注 |
|------|------|------|
| 整地后播种前 | 最优，肥料效果最大化 | 推荐时机 |
| 生长中途 | 次优，效果打折至70% | 紧急补救 |
| 成熟前3天 | 无效（来不及发挥作用） | 不推荐 |

### 6.3 施肥效果公式

```
effective_bonus = base_bonus × timing_modifier

timing_modifier:
    播种时施肥 → 1.0 (100%)
    生长前期（< 25%）→ 0.9 (90%)
    生长中期（25%~75%）→ 0.7 (70%)
    生长后期（> 75%）→ 0.3 (30%)
    成熟前3天 → 0.0 (0%)
```

## 8. 多年生灵茶灌木（TeaBush）

### 8.1 与一年生作物的区别

| 维度 | 一年生作物 | 多年生灵茶灌木 |
|------|:--------:|:-----------:|
| 生命周期 | 一季（收获后铲除/重种） | 全年（可持续采摘） |
| 种植次数 | 每次收获后需重新播种 | 一次种植，长期产出 |
| 采摘频率 | 一次性收获 | 每 3~5 天可采摘一次 |
| 总产量 | 单次高产量 | 长期累积高产量 |
| 投入成本 | 种子成本低 | 灌木苗成本高 |
| 体力消耗 | 每次收获需体力 | 采摘体力极低 |
| 品质 | 受云海影响 | 受云海影响 + 灌木年龄加成 |

### 8.2 灵茶灌木种类

| 灌木 | 苗木成本 | 采摘周期 | 每次产量 | 总产量（季） | 适种季节 |
|------|:-------:|:-------:|:-------:|:----------:|---------|
| 初雾绿茶灌木 | 200G | 5天 | 1个 | ~5个 | 春 |
| 夏云白茶灌木 | 280G | 6天 | 1个 | ~4个 | 夏 |
| 秋霜乌龙灌木 | 350G | 7天 | 1个 | ~4个 | 秋 |
| 冬雪银针灌木 | 420G | 8天 | 1个 | ~3个 | 冬 |

### 8.3 TeaBush 数据结构要点

```cpp
struct TeaBush {
```
