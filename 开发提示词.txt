## 六、关键建议总结

### 6.1 「止血」行动

```
立即停止：无实质产出的功能扩展
立即开始：
1. 购买美术素材包（1周内）
2. 聚焦3位核心NPC内容深耕（持续）
3. 基础音频填充（2周内）
```

### 6.2 「造血」策略

```
核心循环优先：种植→加工→售卖→社交→成长
内容填充次之：对话、事件、作物、配方
系统扩展最后：秘境、多人、Boss
```

### 6.3 评估标准

```
每增加一个系统，必须回答：
1. 它和哪个现有系统形成闭环？
2. 玩家是否能在1-2天内感知到收益？
3. 它是否已经接入存档、日切、UI提示？
4. 它是否可以配置化/数据驱动？
5. 它是否强化了「云海山庄」的差异化？
```
