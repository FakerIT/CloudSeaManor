# Yunhai MVP (C++ First Code)

This is the first runnable C++ prototype for `云海山庄`.

## What it does
- Opens an SFML window
- Runs a game loop at 60 FPS
- Moves a player placeholder rectangle with `WASD`

## Build in Visual Studio (recommended)
1. Open Visual Studio
2. `File -> Open -> Folder...`
3. Select: `d:/coursor/cpp_mvp`
4. Wait for CMake configure and dependency download
5. Select target `yunhai_mvp`
6. Press `F5` to run
